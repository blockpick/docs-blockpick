# 소원/소문 화면

위시 탭에서 접근하는 소원 관련 화면들입니다.

| # | 화면명 | 경로 | 기능 요약 |
|---|--------|------|-----------|
| A-34 | 소원 등록 | `/wish/create` | 한줄평 + 이미지 + 구매URL + 가격 + 카테고리 입력 |
| A-35 | 소원 상세/공감 | `/wish/:id` | 소원 상세, 공감 버튼, 댓글, 공유, 구매 링크 |
| A-36 | 소원 리스트 | `/wish` | 소원 탐색 (인기/최신/마감임박/고액/저액 정렬, 카테고리/가격 필터) |
| A-37 | 내 소원 관리 | `/my/wishes` | 등록한 소원 목록, 공감 현황, 참여자 수, 당첨 횟수 |
| A-38 | 소원 등록자 대시보드 | `/my/wish-dashboard` | 총 공감, 총 참여자, 1+1 혜택 이력 |

## 소원 등록 5단계 (A-34)

1. **URL 입력**: 구매 링크 입력 → 상품 정보 자동 파싱
2. **정보 확인**: 상품명, 가격, 이미지 확인/수정
3. **한줄평 작성**: 소원에 대한 한줄 설명
4. **카테고리 선택**: 9종 카테고리 중 선택
5. **미리보기/확인**: 최종 확인 후 등록

## 소원 리스트 (A-36)

### 카테고리 (9종)
전체, 전자/IT, 패션/뷰티, 식품, 생활, 여행/레저, 문화, 키즈, 기타

### 정렬
- 인기순 (공감 많은 순)
- 최신순 (등록일 기준)
- 마감임박순

### 필터
- 카테고리별
- 가격대별

## 소원 상세 (A-35)

- 상품 정보 (이미지, 가격, 구매 링크)
- 공감 프로그레스바 (목표 공감 대비 현재 공감)
- 댓글 목록
- 상태별 CTA 버튼 (공감하기 / 소문내기 참여 / 마감)
- 업체 경품 표시 (업체 소원인 경우)

## 소문내기 참여 확인

바텀시트로 참여 방식을 선택합니다:
- 10원 참여 (쇼핑캐시 차감)
- 광고 시청 후 무료 참여
- 무료 참여 (이벤트 시)
