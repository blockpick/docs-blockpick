# PRIME 역경매

PRIME 탭의 역경매 화면들입니다.

| # | 화면명 | 경로 | 기능 요약 |
|---|--------|------|-----------|
| A-39 | PRIME 상품 목록 | `/prime` | 역경매 진행 중 상품 리스트, 남은 시간, 참여자 수 |
| A-40-P | PRIME 입찰 | `/prime/:id` | 입찰 금액 입력, 현재 고유 여부 확인, 내 입찰 이력 |
| A-41-P | PRIME 결과 | `/prime/:id/result` | 최저 고유 금액 당첨자 발표, 내 입찰 결과 |

## 역경매 메커닉

1. 운영자가 고가 상품을 등록합니다
2. 사용자가 원하는 가격(1원~상품가)을 입찰합니다
3. 게임 종료 시 **가장 낮으면서 유일한 금액**을 입찰한 사용자가 당첨됩니다
4. 동일 금액을 여러 명이 입찰하면 해당 금액은 무효

## 구현 상태

현재 미구현 상태이며, 기존 가격 예측(OptimalGame) 방식에서 역경매로 전면 변경 예정입니다.
