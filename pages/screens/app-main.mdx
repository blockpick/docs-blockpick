# 앱 메인 탭

5개 탭으로 구성된 메인 네비게이션입니다.

import { Callout } from 'nextra/components'

<Callout type="warning">
  기존 5탭(홈/이벤트/파트너/당첨/MY) → 신규 5탭(홈/데일리/위시/프라임/MY)으로 변경되었습니다.
</Callout>

## 탭 구성

| # | 탭 | 아이콘 | 화면 | 주요 기능 |
|---|-----|--------|------|-----------|
| A-20 | **홈** | 홈 | NewHomeScreen | 배너 캐러셀, 인기 소원 미리보기, 최근 당첨자, 공지, 잔액 표시, 출석체크 |
| A-21 | **데일리** | 해 | DailyScreen | 매일 저가 상품 게임 (커피, 라면 등), **블록픽 판** (블록체인 기반 보드), 오늘의 상품, 참여 현황 |
| A-22 | **위시** | 별 | WishScreen | 소원 리스트(필터/정렬), 소원 등록, **블록픽 판** (블록체인 기반 보드)으로 소문 참여, 업체 소원 |
| A-23 | **프라임** | 다이아 | PrimeScreen | 역경매 상품 목록 (스크롤 리스트 UI), 입찰 화면, 결과 확인 |
| A-24 | **MY** | 사람 | MyScreen | 프로필, 잔액, 내 소원 관리, 당첨 이력, 참여 이력, 설정 |

## 기존 탭 콘텐츠 이동 내역

| 기존 탭 | 기존 콘텐츠 | 이동 위치 |
|---------|------------|-----------|
| 이벤트 (DAILY) | 매일 게임 | **데일리** 탭 (독립) |
| 이벤트 (VIBE) | 소원/소문 게임 | **위시** 탭 (핵심 탭으로 승격) |
| 이벤트 (PRIME) | 역경매 | **프라임** 탭 (독립) |
| 이벤트 (SELECT) | 그리드 선택 | **홀딩** (추후 위시 내 서브 탭 검토) |
| 파트너 | 파트너 브랜드 게임 | **위시** 탭 내 업체 소원으로 통합 |
| 당첨자 | 당첨자 목록/후기 | **홈** 하단 "최근 당첨자" 섹션 |
| 당첨자 | 내 당첨 이력 | **MY** > 당첨 이력 |

## 변경 사유

- SELECT 홀딩
- 소원/소문 메커닉 도입으로 위시 탭 신설
- PRIME 역경매 독립 탭으로 분리 (스크롤 리스트 UI 유지)
- 파트너 게임을 업체 소원으로 통합
- **데일리·위시의 메인 게임 방식을 가챠(좌표 선택)에서 블록픽 판(블록체인 기반 보드)으로 변경**
- **가챠 메커닉은 미니게임 섹션으로 이동**
