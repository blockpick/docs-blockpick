## 금지 상품

소원 등록 시 아래 카테고리에 해당하는 상품은 **등록이 거부**됩니다.

| 카테고리 | 사유 | 탐지 방법 |
|----------|------|-----------|
| **성인용품** | 미성년자 접근 가능 플랫폼 | URL/키워드 필터 |
| **의약품** | 약사법 위반 소지 | URL 블랙리스트 |
| **무기류** | 총포·도검·화약류 등 | 키워드 필터 + 수동 검수 |
| **불법 상품** | 위조품, 밀수품, 해킹 도구 등 | URL 블랙리스트 + 신고 |
| **가상화폐** | 금융 규제, 사행성 우려 | 키워드 필터 |
| **담배/주류** | 미성년자 접근 가능 | URL/키워드 필터 |
| **상품권/현금성 자산** | 현금화 방지 (사행성 대응) | 키워드 필터 + 수동 검수 |

## 소원 등록 검수

### 자동 검수 (등록 즉시)

| 검수 항목 | 규칙 | 실패 시 처리 |
|-----------|------|-------------|
| **URL 유효성** | HTTP 200 응답, 유효한 쇼핑몰 도메인 | 등록 차단, 재입력 요청 |
| **가격 범위** | 100원 ~ 5,000,000원 | 등록 차단, 범위 안내 |
| **이미지 검수 (NSFW)** | AI 이미지 분류 (Google Cloud Vision / AWS Rekognition) | 등록 보류, 수동 검토 큐 |
| **한줄평 비속어** | 비속어 사전 매칭 + 변형어 탐지 | 등록 차단, 수정 요청 |
| **URL 블랙리스트** | 금지 도메인 DB 매칭 | 등록 차단 |

### 수동 검수 (관리자 큐)

자동 검수를 통과했으나 아래 조건에 해당하면 **관리자 검토 큐**에 진입합니다.

- 가격 100만원 이상 고가 상품
- NSFW 탐지 스코어 경계값 (0.5~0.8)
- 신규 등록 유저의 첫 소원
- 신고 이력이 있는 유저의 소원

<Callout type="warning">
  관리자 검수 대기 중인 소원은 **"검수 중" 상태**로 표시되며, 공감 수집은 가능하지만 게임 개설은 불가합니다. 검수 완료까지 최대 24시간을 목표로 합니다.
</Callout>

## 자동 모더레이션

### 키워드 필터

- **비속어 사전**: 한국어 비속어 DB (초성 변형, 특수문자 치환 포함)
- **금지어 사전**: 상품 카테고리별 금지 키워드
- **정규식 패턴**: 전화번호, 개인정보, 외부 링크 패턴 탐지
- **업데이트 주기**: 주 1회 사전 갱신, 신고 기반 즉시 추가

### URL 블랙리스트

- 금지 상품 판매 사이트 도메인 DB 관리
- 피싱/사기 사이트 자동 차단 (Google Safe Browsing API 연동)
- 유저 신고 기반 블랙리스트 추가 프로세스

### 이미지 AI 검수

- **1차**: Google Cloud Vision SafeSearch (adult, violence, racy 탐지)
- **스코어 기준**: 0.8 이상 자동 차단, 0.5~0.8 수동 검토, 0.5 미만 통과
- **2차 (Phase 2)**: 커스텀 모델 학습 (블록픽 특화 부적절 이미지)

## 수동 모더레이션

### 신고 처리 워크플로우

```
유저 신고 접수
  └→ 신고 큐 등록 (자동 분류: 비속어/부적절 상품/사기/기타)
       └→ 관리자 검토 (24시간 내 1차 대응)
            ├→ 정상 → 신고 기각, 신고자에게 결과 알림
            ├→ 경미 → 콘텐츠 수정 요청 + 경고 1회
            ├→ 중대 → 콘텐츠 즉시 삭제 + 경고
            └→ 반복/악의적 → 계정 이용 제한 (7일/30일/영구)
```

### 신고 사유 분류

| 사유 | 설명 | 우선순위 |
|------|------|----------|
| 부적절한 상품 | 금지 카테고리 상품 | 높음 |
| 허위 정보 | 존재하지 않는 상품, 가짜 URL | 높음 |
| 비속어/혐오 표현 | 욕설, 차별 발언 | 중간 |
| 스팸/광고 | 외부 서비스 홍보, 반복 등록 | 중간 |
| 개인정보 노출 | 전화번호, 주소 등 | 높음 |
| 기타 | 위에 해당하지 않는 신고 | 낮음 |

## 댓글 정책

| 규칙 | 내용 |
|------|------|
| **비속어 자동 필터** | 등록 시 실시간 필터링, 마스킹 처리 (예: "시**") |
| **스팸 탐지** | 동일 문구 반복 등록 차단 (1분 내 동일 내용 3회 이상) |
| **신고 3회 자동 숨김** | 서로 다른 유저 3명 이상 신고 시 자동 숨김 처리, 관리자 검토 큐 등록 |
| **외부 링크 차단** | 댓글 내 URL 등록 불가 (피싱/스팸 방지) |
| **길이 제한** | 최대 200자 |

## 이의 제기

<Callout type="info">
  제재를 받은 유저는 공정한 이의 제기 절차를 이용할 수 있습니다.
</Callout>

| 항목 | 내용 |
|------|------|
| **이의 제기 기한** | 제재 통보일로부터 **7일 이내** |
| **접수 방법** | 앱 내 고객센터 → "제재 이의 제기" 메뉴 |
| **처리 기한** | 접수 후 **3영업일 이내** 결과 통보 |
| **검토 기준** | 원본 콘텐츠, 신고 내역, 유저 이력 종합 검토 |
| **결과** | 제재 유지 / 제재 완화 / 제재 철회 |
| **재이의** | 동일 건에 대해 1회 추가 이의 가능 (최종 심사) |

### 제재 단계

| 단계 | 조건 | 제재 내용 |
|------|------|-----------|
| 1단계 (경고) | 첫 번째 위반 | 경고 알림 + 콘텐츠 수정/삭제 |
| 2단계 (일시 제한) | 경고 누적 3회 | 7일 소원 등록/댓글 제한 |
| 3단계 (장기 제한) | 경고 누적 5회 | 30일 서비스 이용 제한 |
| 4단계 (영구 차단) | 심각한 위반 또는 반복 위반 | 영구 이용 정지 |
