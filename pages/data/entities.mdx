# 데이터 엔티티

블록픽의 핵심 데이터 엔티티 정의입니다.

## User (사용자)

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | ID | O | 고유 식별자 |
| email | String | O | 이메일 (로그인 ID) |
| nickname | String | - | 표시 닉네임 |
| phone | String | - | 전화번호 (인증 완료) |
| passwordHash | String | O | 비밀번호 해시 |
| provider | Enum | O | 가입 경로 (EMAIL/GOOGLE/APPLE) |
| walletAddress | String | - | Ronin 지갑 주소 |
| shoppingCash | Int | O | 쇼핑캐시 잔액 |
| eventPoints | Int | O | 이벤트 포인트 잔액 |
| status | Enum | O | 계정 상태 (ACTIVE/SUSPENDED/WITHDRAWN) |
| marketingConsent | Boolean | O | 마케팅 수신 동의 |
| createdAt | DateTime | O | 가입일 |
| updatedAt | DateTime | O | 수정일 |

## Game (게임)

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | ID | O | 고유 식별자 |
| title | String | O | 게임 제목 |
| description | String | - | 게임 설명 |
| gameType | Enum | O | DAILY / SELECT / VIBE / PRIME / PARTNER |
| status | Enum | O | SCHEDULED / ACTIVE / ENDED / CANCELLED |
| entryFee | Int | O | 참여비 (쇼핑캐시) |
| maxEntries | Int | O | 최대 참여 수 |
| currentEntries | Int | O | 현재 참여 수 |
| gridRows | Int | O | 그리드 행 수 |
| gridCols | Int | O | 그리드 열 수 |
| startTime | DateTime | O | 게임 시작 시간 |
| endTime | DateTime | O | 게임 종료 시간 |
| onchainTxHash | String | - | 블록체인 트랜잭션 해시 |
| onchainContractAddr | String | - | 스마트 컨트랙트 주소 |
| createdAt | DateTime | O | 생성일 |

|------|------|------|------|

| gameId | ID | O | 소속 게임 ID |

## Entry (게임 참여)

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| id | ID | O | 고유 식별자 (엔트리 ID) |
| gameId | ID | O | 게임 ID |
| userId | ID | - | 회원 ID (비회원은 null) |
| guestToken | String | - | 비회원 토큰 |
| phoneNumber | String | - | 비회원 전화번호 |
| gridX | Int | O | 선택한 X좌표 (열) |
| gridY | Int | O | 선택한 Y좌표 (행) |
| encryptedCoord | String | - | 암호화된 좌표 데이터 |
| coordHash | String | - | 좌표 SHA-256 해시 |
| status | Enum | O | PENDING / CONFIRMED / WON / LOST |
| onchainTxHash | String | - | 블록체인 트랜잭션 해시 |
| idempotencyKey | String | - | 중복 참여 방지 키 |
| createdAt | DateTime | O | 참여 시간 |
