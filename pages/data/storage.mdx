# 저장소

데이터 저장소별 용도입니다.

## 저장소 분류

| 저장소 | 위치 | 용도 |
|--------|------|------|
| **서버 DB** | 백엔드 | 사용자, 게임, 엔트리, 상품, 거래 등 핵심 데이터 |
| **AWS S3** | 클라우드 | 상품 이미지, 후기 이미지 |
| **Ronin 블록체인** | 분산 네트워크 | 좌표 암호화 데이터, 트랜잭션 해시, 게임 결과 |
| **Hive (앱)** | 디바이스 로컬 | 캐시 데이터, 오프라인 임시 저장 |
| **Secure Storage (앱)** | 디바이스 Keychain/Keystore | JWT 토큰, 지갑 프라이빗 키 |
| **SharedPreferences (앱)** | 디바이스 로컬 | 사용자 설정, 튜토리얼 상태 |
| **localStorage (웹)** | 브라우저 | JWT 토큰, 게스트 토큰 |
