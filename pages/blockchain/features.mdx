# 주요 기능

블록체인을 활용한 핵심 기능들입니다.

## 기능 목록

| 기능 | 설명 |
|------|------|
| **지갑 자동 생성** | 회원가입/로그인 시 Ethereum 지갑 자동 생성 |
| **키 암호화** | AES-256으로 좌표 암호화, 백엔드는 SHA-256 해시만 보유 |
| **스마트 컨트랙트** | 암호화 키 요청/발급, 추첨 결과 기록 |
| **트랜잭션 추적** | 참여 엔트리의 온체인 확인 상태 폴링 |
| **투명성** | 트랜잭션 해시(`onchainTxHash`) 및 컨트랙트 주소(`onchainContractAddr`) 공개 |

## 지갑 생성 플로우

1. BIP39 니모닉 시드 문구 생성
2. 니모닉에서 프라이빗 키 파생
3. 프라이빗 키에서 지갑 주소 생성
4. 프라이빗 키를 Flutter Secure Storage에 암호화 저장

## 좌표 암호화 플로우

1. 사용자가 좌표를 선택합니다
2. 좌표를 AES-256으로 암호화합니다
3. 암호화된 데이터를 블록체인에 기록합니다
4. SHA-256 해시를 백엔드에 전달합니다
5. 게임 종료 시까지 백엔드도 원본 좌표를 알 수 없습니다

## 블록체인 API

| 오퍼레이션 | 타입 | 입력 | 출력 |
|-----------|------|------|------|
| `requestEncryptionKey` | Mutation | - | keyId |
| `getEncryptionKeyStatus` | Query | keyId | status |
| `getEntryStatus` | Query | entryId | onchainStatus |
