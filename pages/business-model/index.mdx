# 수익 모델 총론

블록픽은 **포인트(P) 기반 공정 추첨 커머스** 플랫폼으로, 다섯 가지 수익원이 상호 보완적으로 작동하는 복합 수익 구조를 갖습니다. 유저가 포인트를 소비하면서 발생하는 참여비, 포인트 충전 결제, 광고 노출, B2B 파트너 광고, PRIME 역경매 수수료가 핵심 수익원입니다.

---

## 1. 수익 구조 개요

```
블록픽 수익 = 게임 참여비 + 인앱 결제 + 리워드 광고 + B2B 파트너 + PRIME 수수료
```

| # | 수익원 | 과금 주체 | 유형 | 예상 비중 (성숙기) |
|---|--------|-----------|------|------------------|
| 1 | 게임 참여비 (포인트 소비) | 유저 | 직접 수익 | 35% |
| 2 | 인앱 결제 (포인트 충전) | 유저 | 직접 수익 | 30% |
| 3 | 리워드 광고 (AdMob) | 광고주 (Google) | 간접 수익 | 15% |
| 4 | 업체 소원 / 파트너 광고 | 파트너사 (B2B) | 직접 수익 | 15% |
| 5 | PRIME 역경매 수수료 | 유저 / 낙찰자 | 직접 수익 | 5% |

<Callout type="info">
초기(Phase 1)에는 **리워드 광고 + 소액 참여비**가 주 수익원이며, MAU 성장에 따라 인앱 결제와 B2B 파트너 비중이 점차 확대됩니다.
</Callout>

---

## 2. 수익원별 상세

### 2-1. 게임 참여비 수익

유저가 게임에 참여할 때 포인트(P)를 소비합니다. 소비된 포인트의 일부는 상품 조달 비용으로 사용되고, 나머지가 수익으로 남습니다.

| 게임 유형 | 참여비 | 메커니즘 |
|-----------|--------|----------|
| DAILY (데일리 추첨) | 1P ~ 10P | 하루 1회, 저비용 참여 유도 |
| WISH (소원) | 10P | 소원 게시 + 참여 |
| VIBE (소문내기) | 10P | SNS 공유 조건 참여 |
| PRIME (역경매) | 입찰가 전액 | 최저가 입찰자 낙찰 |

**수익 공식:**
```
참여비 수익 = Σ(게임별 참여비 × 참여자 수) - 당첨 상품 원가
```

**성장 드라이버:**
- DAU 증가 → 참여 빈도 증가
- 게임 다양화 → 포인트 소비 채널 확대
- 무료 포인트 소멸(90일) → 유료 포인트 전환 유도

---

### 2-2. 인앱 결제 수익 (포인트 충전)

유저가 현금으로 포인트를 충전합니다. Apple IAP / Google Play Billing을 통해 결제가 이루어지며, 플랫폼 수수료 차감 후 순수익이 발생합니다.

| 충전 패키지 | 금액 | 제공 포인트 | 보너스 |
|------------|------|------------|--------|
| 스타터 | 1,100원 | 100P | - |
| 베이직 | 5,500원 | 500P | +50P (10%) |
| 스탠다드 | 11,000원 | 1,000P | +150P (15%) |
| 프리미엄 | 33,000원 | 3,000P | +600P (20%) |
| VIP | 110,000원 | 10,000P | +2,500P (25%) |

**수익 공식:**
```
인앱 결제 순수익 = 결제액 × (1 - 플랫폼 수수료율) × 전환율
```

**성장 드라이버:**
- 무료 포인트 소진 유저의 유료 전환
- 희귀 상품 / 한정 게임으로 구매 동기 부여
- 충전 보너스로 대량 구매 유도

---

### 2-3. 리워드 광고 수익 (AdMob)

Google AdMob 기반 리워드 광고로, 유저가 광고를 시청하면 무료 포인트를 지급받습니다. 블록픽은 광고 노출당 수익(eCPM)을 획득합니다.

| 구분 | 내용 |
|------|------|
| 광고 유형 | 리워드 동영상 (30초) |
| 일일 최소 노출 | 11회 고정 (유저당) |
| 포인트 지급 | 시청 완료 시 1P ~ 5P |
| 수익 기준 | eCPM 기반 (한국 평균 ₩5,000~₩15,000/1,000회) |
| 포인트 성격 | 무료 획득 (90일 소멸) |

**수익 공식:**
```
광고 수익 = DAU × 일평균 광고 시청 횟수 × eCPM / 1,000
```

**성장 드라이버:**
- DAU 증가 → 노출 수 증가
- 광고 시청 = 무료 포인트 획득 동기 → 높은 시청 완료율
- AdMob Mediation으로 eCPM 최적화

---

### 2-4. 업체 소원 / 파트너 광고 수익 (B2B)

브랜드/업체가 직접 게임(소원)을 개설하고 광고비를 지불합니다. 자사 상품을 당첨 상품으로 제공하여 브랜드 홍보 + 유저 획득을 동시에 달성합니다.

| 광고 유형 | 설명 | 과금 방식 |
|----------|------|----------|
| 업체 소원 | 브랜드 상품을 당첨품으로 게임 개설 | 게임 개설비 + 노출비 |
| 스폰서 배너 | 게임 목록 상단 노출 | CPM / 고정 패키지 |
| 브랜드 미니게임 | 자사 브랜드 테마 미니게임 | 커스터마이징 비용 |

**수익 공식:**
```
B2B 수익 = 파트너 수 × 평균 월 계약금액
```

---

### 2-5. PRIME 역경매 수수료

PRIME 게임에서 유저들이 입찰에 참여하며 납부한 포인트가 수익이 됩니다. 최저 고유 입찰가 낙찰 방식으로, 대부분의 참여비가 수익으로 귀속됩니다.

| 항목 | 내용 |
|------|------|
| 수익 구조 | 전체 참여비 - 상품 원가 - 낙찰자 환급 |
| 참여비 성격 | 소비형 (환급 없음, 단 낙찰자 제외) |
| 예상 마진율 | 30% ~ 60% (상품 원가에 따라 상이) |

---

## 3. 비용 구조

```
총 비용 = 상품 조달 + 인프라 + 마케팅/UA + 플랫폼 수수료 + 인건비
```

| 비용 항목 | 세부 내용 | 예상 비율 |
|----------|----------|----------|
| 상품 조달 | 쿠폰, 기프티콘, 실물 배송 상품 | 40% |
| 서버 / 블록체인 인프라 | AWS + 블록체인 노드 운영 | 10% |
| 마케팅 / UA | 퍼포먼스 광고, 인플루언서, CPI | 25% |
| 플랫폼 수수료 | Apple 30% / Google 15~30% (인앱 결제 기준) | 15% |
| 인건비 | 개발, 운영, CS | 10% |

<Callout type="warning">
**플랫폼 수수료 주의:** Apple App Store는 인앱 결제의 30%를 수수료로 가져갑니다. Google Play는 첫 100만 달러까지 15%, 초과분은 30%입니다. 실질 포인트 충전 순이익률은 약 65~70%입니다.
</Callout>

## 5. Phase별 수익 전략

### Phase 1: 트래픽 확보 (MAU 1만 이하)

**핵심 전략:** 무료 유저 기반 확대 → 광고 수익 극대화

| 수익원 | 전략 |
|--------|------|
| 리워드 광고 | 일일 11회 고정 노출, 미니게임 보상으로 시청률 극대화 |
| 게임 참여비 | 무료 포인트로 참여 유도, 소액 참여비 설정 |
| 인앱 결제 | 첫 충전 보너스, 패키지 할인으로 전환 유도 |
| B2B | 파일럿 파트너 1~3개 확보, 케이스 스터디 구축 |

**목표 지표:**
- DAU/MAU 비율: 30% 이상
- 광고 시청 완료율: 80% 이상
- 인앱 결제 전환율: 3%

---

### Phase 2: 수익화 가속 (MAU 1만 ~ 10만)

**핵심 전략:** 유료 전환율 향상 + B2B 파트너 확대

| 수익원 | 전략 |
|--------|------|
| 인앱 결제 | 프리미엄 게임 / 한정 상품으로 결제 동기 강화 |
| B2B 파트너 | 정형화된 광고 패키지 출시, 영업팀 운영 |
| PRIME | 고가 상품 역경매로 참여비 규모 확대 |
| 리워드 광고 | AdMob Mediation 도입, eCPM 최적화 |

**목표 지표:**
- ARPPU: ₩20,000 이상
- B2B 파트너: 10개 이상
- LTV/CAC: 3:1 이상

---

### Phase 3: 플랫폼 도약 (MAU 10만 이상)

**핵심 전략:** B2B 광고 플랫폼화 + 프리미엄 기능 출시

| 수익원 | 전략 |
|--------|------|
| B2B 광고 플랫폼 | 셀프서브 광고 집행 도구 오픈, CPM/CPC 자동 경매 |
| 프리미엄 멤버십 | 월정액 구독으로 광고 제거 + 포인트 보너스 |
| 데이터 사업 | 구매 의향 데이터 집계 → 시장 인사이트 보고서 |
| 글로벌 확장 | 동남아 시장 진출, 현지 파트너 광고 유치 |

**목표 지표:**
- B2B 광고 매출 비중: 30% 이상
- 월정액 구독자: MAU의 5%
- 글로벌 MAU: 전체 30% 이상

<Callout type="warning">
Phase 3 진입 전 블록체인 공정성 인증 / 규제 준수 체계를 완비해야 글로벌 확장이 가능합니다.
</Callout>
