# 수익 모델

블록픽의 4가지 수익 모델입니다.

## 수익 구조

| 모델 | 설명 | 구현 상태 |
|------|------|-----------|
| **게임 참여비** | 쇼핑캐시로 게임 참여 시 차감 | 완료 |
| **인앱 결제** | 쇼핑캐시 충전 | 완료 (충전 화면) |
| **리워드 광고** | Google AdMob 리워드 동영상 광고 | 완료 |
| **파트너 협업** | 브랜드 파트너 게임 | 완료 |

## 게임 참여비

- 사용자가 게임에 참여할 때 쇼핑캐시가 차감됩니다
- 게임 타입별로 참여비가 다릅니다
- DAILY: 저가, VIBE(소문내기): 10원, PRIME: 입찰 금액

## 인앱 결제

- 사용자가 쇼핑캐시를 유료로 충전합니다
- PG사 연동은 Phase 1에서 개발 예정입니다
- Apple IAP / Google Play Billing 대응이 필요합니다

## 리워드 광고

- Google AdMob 리워드 동영상 광고를 시청하면 이벤트 포인트를 지급합니다
- 일일 시청 횟수 제한이 있습니다

## 파트너 협업

- 외부 브랜드가 자사 상품으로 게임을 개설합니다
- 비회원도 참여 가능하여 브랜드 홍보 효과가 있습니다
- 업체 소원으로 통합 운영 예정입니다

## 신규 수익 모델 (예정)

- **업체 소원 광고**: 업체가 소원을 등록하고 광고비를 지불
- **PRIME 역경매**: 입찰 참여비 수익
- **드롭쉬핑**: 당첨 상품의 구매 대행 마진
