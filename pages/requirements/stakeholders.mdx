# 이해관계자

## 사용자 역할

| 역할 | 설명 | 접근 채널 |
|------|------|-----------|
| **일반 회원** | 이메일/소셜 가입 후 게임 참여 | 앱, 웹 |
| **비회원 (게스트)** | 파트너 게임에 전화번호로 참여 | 웹 |
| **관리자** | 게임/상품/사용자 관리 | admin-blockpick |
| **파트너 브랜드** | 자사 상품으로 게임 개설 | admin |

## 페르소나

### 김민수 (27세, 직장인)
- 모바일 게임에 익숙, 출퇴근 시간 앱 사용
- 소액으로 고가 상품 당첨 희망
- 매일 출석체크, DAILY 게임 참여, 리워드 광고 시청

### 이수진 (33세, 프리랜서)
- SNS 활발, 새 앱 탐색
- 재미있는 게임, 당첨 후기 공유
- SELECT/VIBE 선호, 미니게임

### 박지현 (45세, 주부)
- 모바일 앱 익숙하지 않음, 실용적 상품 관심
- 생활용품/식품 당첨 희망
- 파트너 게임, 간단한 가챠 선호

## 사용자 규모 목표

| 단계 | 사용자 수 | 시기 |
|------|-----------|------|
| Phase 1 (MVP) | 1,000~5,000명 | 서비스 런칭 |
| Phase 2 (성장) | 5,000~30,000명 | 런칭 후 3~6개월 |
| Phase 3 (확장) | 30,000명+ | 런칭 후 6~12개월 |
