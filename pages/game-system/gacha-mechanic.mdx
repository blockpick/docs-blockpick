# GachaCanvas 메커닉 (미니게임)

import { Callout } from 'nextra/components'

<Callout type="warning">
  가챠 메커닉은 더 이상 메인 게임 방식이 아닙니다. 데일리·위시의 메인 게임은 **블록픽 판** (블록체인 기반 보드)으로 변경되었으며, 가챠는 **미니게임**으로 이동되었습니다.
</Callout>

미니게임 중 하나로 제공되는 가챠(좌표 선택) 메커닉입니다.

## 2단계 좌표 선택

1. **X좌표 선택 (COL)**: 애니메이션 크로스헤어가 수평 이동 → 탭으로 정지
2. **Y좌표 선택 (ROW)**: 크로스헤어가 수직 이동 → 탭으로 정지
3. **결과 확인**: 선택된 좌표의 당첨 여부 판정
4. **이펙트**: 파티클 시스템 (컨페티, 스파클), Elastic 이징 애니메이션

## 기술 구현

- **컴포넌트**: `GachaCanvas.tsx` (웹)
- **애니메이션**: Framer Motion (웹), Flutter Animate (앱)
- **파티클**: 커스텀 Canvas 기반 파티클 시스템
- **이징**: Elastic 이징으로 크로스헤어 정지 효과

## 앱에서의 사용

앱에서는 WebView를 통해 웹의 GachaCanvas를 로드합니다. WebView Bridge를 통해 앱과 웹이 양방향으로 통신합니다.
