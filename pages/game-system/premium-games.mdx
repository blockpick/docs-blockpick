# 프리미엄 게임

웹에서 제공되는 심화 스토리/심리학 메커닉 기반 고급 게임입니다.

## Singularity (우주 테마)

### 기술 스택
- Three.js + React Three Fiber + Post-processing

### 5단계 플로우
1. **성운(Nebula) 선택**: 우주 배경에서 성운을 선택
2. **발사대(LaunchPad) 준비**: 발사 준비 단계
3. **우주 필드(CosmicField) 탐색**: 우주 공간 탐색
4. **중력/질량 메커닉**: 물리 법칙 기반 인터랙션
5. **결과 확인**: 최종 좌표 결정 및 당첨 판정

### 3D 요소
- StarfieldBG (별 배경)
- StellarSeed (항성 시드)
- DustField (먼지 필드)
- Shockwave (충격파)

### UI 요소
- MassGauge (질량 게이지)
- GravitySlider (중력 슬라이더)
- CosmicEventAlert (우주 이벤트 알림)

## Abyssal (심해 테마)

### 기술 스택
- Three.js + React Three Fiber + Rapier Physics

### 14종 심리학 메커닉
다양한 심리학 이론을 게임 메커닉에 적용한 심해 탐험 게임입니다.

### 심도 레이어
1. 표층 (Sunlight)
2. 박명대 (Twilight)
3. 자정대 (Midnight)
4. 심연대 (Abyssal)
5. 초심해대 (Hadal)

### 시스템
- **산소(O2) 관리**: 탐험 시간 제한
- **위협 시스템**: 심해 생물 위협 관리
- **딥 이벤트**: 심도별 특수 이벤트

### 3D 요소
- OceanBackground
- LureObject (미끼)
- TreasureObject (보물)
- PredatorEntity (포식자)
- GodRaysEffect (빛줄기)

### UI 요소
- DepthGauge (심도 게이지)
- BuoyancyTank (부력 탱크)
- LuminanceSlider (발광 슬라이더)
- ThreatIndicator (위협 표시기)
