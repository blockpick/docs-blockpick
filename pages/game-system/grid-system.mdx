# 그리드 시스템

앱에서 구현된 대규모 그리드 기반 게임 시스템입니다.

import { Callout } from 'nextra/components'

<Callout type="warning">
  SELECT 게임 타입은 현재 홀딩 상태이지만, 그리드 시스템은 구현 완료되어 있습니다.
</Callout>

## 스펙

- 최대 **1000 x 1000** 그리드 지원 (100만 셀)
- **CustomPaint** 기반 렌더링 (Flutter, 성능 최적화)
- **핀치 줌 / 팬 제스처** 지원
- **LOD (Level of Detail)**: 줌 레벨에 따른 디테일 조절
- **미니맵**: 전체 그리드 위치 파악

## 성능 최적화

| 기법 | 설명 |
|------|------|
| LOD | 줌 아웃 시 셀 디테일 감소, 줌 인 시 상세 표시 |
| 뷰포트 렌더링 | 화면에 보이는 셀만 렌더링 |
| CustomPaint | Flutter Canvas API 직접 사용 (위젯 트리 오버헤드 없음) |

## 성능 기준 (NFR-002)

- 일반 화면: 60fps
- 그리드 (1000x1000): 30fps 이상
- 가챠 애니메이션: 60fps
