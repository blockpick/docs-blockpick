# 소원/소문 시스템

사용자가 원하는 상품을 등록하고, 다른 사용자들이 참여하는 시스템입니다.

## 핵심 개념

- **소원**: 사용자가 갖고 싶은 상품을 등록하는 것
- **소문내기**: 소원에 10원으로 참여하는 추첨 게임
- **공감**: 소원에 관심을 표시하는 행위 (공감 목표 달성 시 게임 개설)
- **업체 소원**: 기업이 등록하는 프로모션 상품

## 소원 등록 플로우

1. 구매 URL 입력 (자동 상품 정보 파싱)
2. 상품 정보 확인/수정 (이름, 가격, 이미지)
3. 한줄평 작성
4. 카테고리 선택 (9종)
5. 미리보기 및 확인

## 소문내기 참여

| 항목 | 내용 |
|------|------|
| **참여비** | 10원 (쇼핑캐시) |
| **무료 참여** | 리워드 광고 시청 또는 이벤트 시 |
| **게임 방식** | 가챠 스타일 (VIBE 타입) |
| **당첨** | 1등 상품 + 소원 등록자 1+1 혜택 |

## 공감 시스템

- 사용자가 소원에 공감 버튼을 누릅니다
- 공감 프로그레스바로 목표 대비 현재 공감 수를 표시합니다
- 일정 공감 수 달성 시 소문내기 게임이 개설됩니다

## 업체 소원

- 기업이 자사 상품을 소원으로 등록하는 프로모션
- BRAND 배지로 구분
- 관리자(ADM-03)가 대리 등록
- 광고 패키지 설정 가능 (노출 위치, 기간, 예산)
