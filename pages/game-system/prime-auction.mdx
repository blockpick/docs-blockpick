# PRIME 역경매

최저 고유가 입찰 방식의 프리미엄 경매 시스템입니다.

## 메커닉

1. 운영자가 고가 상품을 등록합니다
2. 사용자가 원하는 가격(1원~상품가)을 입찰합니다
3. 게임 종료 시 **가장 낮으면서 유일한 금액**을 입찰한 사용자가 당첨됩니다
4. 동일 금액을 여러 명이 입찰하면 해당 금액은 무효

## 화면 구성

| 화면 | 경로 | 기능 |
|------|------|------|
| 상품 목록 | `/prime` | 역경매 진행 중 상품, 남은 시간, 참여자 수 |
| 입찰 | `/prime/:id` | 금액 입력, 고유 여부 확인, 내 입찰 이력 |
| 결과 | `/prime/:id/result` | 최저 고유 금액 당첨자 발표 |

## 구현 상태

현재 미구현 상태입니다. 기존 가격 예측(OptimalGame) 방식에서 역경매로 전면 변경 예정입니다.
