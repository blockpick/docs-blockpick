# 미니게임 (30종+)

웹 기반으로 제공되는 다양한 미니게임입니다.

## 물리 시뮬레이션 계열

| 게임 | 설명 | 엔진 |
|------|------|------|
| **Plinko** | 공 낙하 물리 시뮬레이션 | Matter.js |
| **CoinPusher** | 코인 푸셔 (2D) | Matter.js |
| **CoinPusher3D** | 코인 푸셔 (3D) | Three.js |
| **BallHole** | 공 홀인원 (2D) | Matter.js |
| **BallHole3D** | 공 홀인원 (3D) | Three.js |
| **BottleFlip** | 물병 뒤집기 | Matter.js |
| **BlockPull** | 젠가 스타일 블록 빼기 | Matter.js |
| **FreeThrow** | 농구 자유투 | Matter.js |

## 스킬/타이밍 계열

| 게임 | 설명 |
|------|------|
| **StackTower / StackTower3D** | 블록 쌓기 (타이밍) |
| **SwingStopper / SwingStopper3D** | 스윙 정지 (타이밍) |
| **ClawMachine** | 크레인 인형뽑기 |
| **CapsuleMachine** | 캡슐 머신 (슬롯) |
| **Roulette** | 룰렛 (스핀) |
| **Edge** | 줄 당기기 장력 게임 |
| **PourPerfect** | 정밀 따르기 |
| **DalgonaGame** | 달고나 모양 깎기 |
