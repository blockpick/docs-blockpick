# 서비스 소개

**블록픽(BlockPick)** 은 블록체인 기반의 공정한 추첨형 게임 플랫폼으로, 사용자가 다양한 인터랙션 방식으로 좌표를 선택하여 상품에 응모하는 서비스입니다.

## 서비스 정의

온라인 경품/추첨 시장은 투명성과 공정성에 대한 사용자 신뢰가 핵심 과제입니다. 기존 추첨 서비스는 결과 조작 의혹, 불투명한 당첨 확률, 단조로운 참여 경험 등의 문제를 안고 있습니다.

블록픽은 블록체인 기술을 활용하여 추첨 결과의 투명성을 보장하고, 30종 이상의 다양한 인터랙티브 게임 메커닉으로 사용자에게 재미있는 참여 경험을 제공하는 차세대 추첨형 게임 플랫폼입니다.

## 서비스 범위

### 범위 내 (In Scope)

| 영역 | 포함 범위 |
|------|-----------|
| **모바일 앱** | iOS/Android Flutter 앱 - 메인 사용자 경험, 게임 참여, 회원 관리, 결제, 블록체인 |
| **웹 서비스** | Next.js 웹 - 랜딩 페이지, WebView 게임 화면, 미니게임, 프리미엄 3D 게임 |
| **백엔드 API** | GraphQL API - 인증, 게임 관리, 결제, 블록체인 연동 |
| **블록체인** | Ronin 체인 스마트 컨트랙트 - 추첨 결과 기록, 좌표 암호화 |
| **관리자 시스템** | admin-blockpick - 게임/상품/사용자 관리 |

### 범위 밖 (Out of Scope)

| 항목 | 사유 |
|------|------|
| 자체 결제 PG 연동 | 현재 캐시 충전 화면만 구현, PG사 연동은 별도 프로젝트 |
| 푸시 알림 (FCM/APNs) | 앱 내 알림 화면은 있으나 푸시 인프라는 미구현 |
| 실시간 채팅/CS 시스템 | 고객센터 화면은 존재하나 실시간 연동 없음 |
| 다국어(i18n) | 현재 한국어 단일 언어 |
| 블록체인 지갑 외부 연동 | MetaMask 등 외부 지갑 연동 없음 (자동 생성 지갑만) |
