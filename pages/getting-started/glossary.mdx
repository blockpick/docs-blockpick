# 용어 정의

블록픽 서비스에서 사용하는 주요 용어입니다.

| 용어 | 정의 |
|------|------|
| **블록(Block)** | 게임 그리드에서 사용자가 선택할 수 있는 최소 단위 (셀) |
| **픽(Pick)** | 사용자가 블록을 선택하는 행위 |
| **가챠(Gacha)** | 2단계(X좌표 -> Y좌표) 애니메이션 기반 좌표 선택 방식 |
| **그리드(Grid)** | 행(Row) x 열(Col)로 구성된 게임 판 (최대 1000x1000) |
| **엔트리(Entry)** | 게임에 대한 1회 참여 기록 |
| **쇼핑캐시** | 게임 참여에 사용하는 유료 재화 (충전 필요) |
| **이벤트 포인트** | 활동으로 획득하는 무료 재화 |
| **온체인(On-chain)** | 블록체인에 기록된 트랜잭션/데이터 |
| **Ronin** | 블록픽이 사용하는 블록체인 네트워크 (Ethereum 호환) |
| **Giftshow** | 기프티쇼 - 모바일 쿠폰 발급 플랫폼 |
| **WebView Bridge** | Flutter 앱과 웹 게임 화면 간 통신 인터페이스 |
| **크리에이티브 픽 모드** | 센서/카메라/음성 등 다양한 입력을 활용한 좌표 선택 방식 |
| **미니게임** | 물리엔진/3D 기반 캐주얼 게임 (웹) |
| **프리미엄 게임** | 심화 스토리/심리학 메커닉을 적용한 고급 게임 |
| **파트너 게임** | 외부 브랜드와 협업하여 운영하는 게임 (비회원 참여 가능) |
| **소원(Wish)** | 사용자가 갖고 싶은 상품을 등록하는 기능 |
| **소문(Rumor)** | 소원에 10원으로 참여하는 추첨 게임 |
| **PRIME 역경매** | 최저 고유가 입찰 방식의 프리미엄 경매 시스템 |
