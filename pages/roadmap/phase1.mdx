# Phase 1 - MVP

핵심 기능 완성 및 서비스 런칭 단계입니다.

## 목표
- 핵심 게임 참여 플로우 완성
- 실결제 연동 (PG사)
- App Store / Google Play 출시

## 포함 기능

| 분류 | 기능 | 상태 | 작업 |
|------|------|------|------|
| 인증 | 이메일/소셜/SMS | 완료 | 안정화 |
| 게임 | 가챠/그리드/옵티멀 | 완료 | 버그 수정 |
| 게임 | 파트너/즉시 당첨 | 완료 | 안정화 |
| 블록체인 | 지갑/암호화/TX | 완료 | 에러 복구 |
| 결제 | PG 충전 | 미구현 | **신규 개발** |
| 보안 | API 보안 | 부분 | HTTPS, Rate Limit |
