# 로드맵 개요

3단계 개발 로드맵입니다.

| 단계 | 목표 | 기간 | 예상 사용자 |
|------|------|------|-------------|
| **Phase 1 - MVP** | 핵심 기능 안정화, PG 결제, 스토어 출시 | 약 3개월 | 1,000~5,000명 |
| **Phase 2 - 성장** | 리텐션 강화, 보안 고도화, 성능 최적화 | 약 3개월 | 5,000~30,000명 |
| **Phase 3 - 확장** | 글로벌, 새 게임, 접근성 강화 | 약 4개월 | 30,000명+ |

## Phase 간 의존성

- Phase 1 → Phase 2: 결제 안정화, 스토어 출시 완료 필수
- Phase 2 → Phase 3: 보안 감사 통과, 성능 기준 달성, 모니터링 구축 필수
