# 신규 사용자 흐름

처음 블록픽을 사용하는 사용자의 흐름입니다.

## 기본 흐름

```
앱 실행 → 스플래시 → 권한 요청 → 온보딩 → 회원가입 → 홈 화면
```

## 단계별 설명

### 1. 앱 실행 및 스플래시

앱 실행 시 스플래시 화면이 표시되며, 네트워크 연결 상태를 확인합니다.

### 2. 권한 요청

크리에이티브 픽 모드를 위한 선택적 권한을 요청합니다:
- 위치 권한 (선택)
- 카메라 권한 (선택)
- 마이크 권한 (선택)

### 3. 온보딩

서비스 소개 슬라이드가 표시됩니다. 첫 사용자에게만 노출됩니다.

### 4. 회원가입

4가지 가입 방식 중 선택할 수 있습니다:

| 방식 | 플로우 |
|------|--------|
| **이메일/비밀번호** | 이메일 입력 → 인증코드 발송 → 코드 확인 → 비밀번호 설정 → 약관 동의 → 가입 완료 |
| **Google 소셜 로그인** | Google OAuth → 전화번호 인증 → 약관 동의 → 가입 완료 |
| **Apple 소셜 로그인** | Apple ID 인증 → 전화번호 인증 → 약관 동의 → 가입 완료 |
| **전화번호 인증** | SMS 인증코드 → 이메일/비밀번호 설정 → 약관 동의 |

### 5. 홈 화면 진입

가입 완료 후 자동 로그인되어 홈 화면으로 이동합니다.
