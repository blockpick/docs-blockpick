# 기존 사용자 흐름

이미 가입된 사용자의 흐름입니다.

## 기본 흐름

```
앱 실행 → 스플래시 → 자동 로그인(토큰 복원) → 홈 화면
```

## 자동 로그인

- Flutter Secure Storage에 저장된 `refreshToken`을 이용하여 자동으로 `accessToken`을 갱신합니다
- 토큰이 유효하면 스플래시 화면 후 바로 홈 화면으로 진입합니다
- `refreshToken`이 만료된 경우 로그인 화면으로 이동합니다

## 토큰 갱신 플로우

1. 앱 시작 시 저장된 `refreshToken` 확인
2. `refreshToken` Mutation 호출
3. 새로운 `accessToken` + `refreshToken` 발급
4. 홈 화면 진입
