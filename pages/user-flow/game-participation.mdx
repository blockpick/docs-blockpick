# 게임 참여 흐름

사용자가 게임에 참여하는 전체 흐름입니다.

## 기본 흐름

```
게임 선택 → 게임 상세 → 좌표 선택(가챠/그리드/옵티멀) → 참여 확인
→ 블록체인 트랜잭션 → 결과 확인 → (당첨 시) 상품 수령
```

## 상세 단계

### 1. 게임 선택

데일리/위시/프라임 탭에서 참여할 게임을 선택합니다.

### 2. 게임 상세

상품 정보, 남은 시간, 참여 현황 등을 확인하고 "참여하기" 버튼을 탭합니다.

### 3. 좌표 선택

게임 타입에 따라 다른 방식으로 좌표를 선택합니다:

| 게임 타입 | 좌표 선택 방식 |
|-----------|----------------|
| DAILY | 가챠 스타일 (X좌표 → Y좌표 애니메이션) |
| VIBE (소원/소문) | 가챠 스타일 (소원 상품 이미지 배경) |
| PRIME | 역경매 입찰 금액 입력 |
| SELECT | 그리드에서 직접 블록 터치 (홀딩 중) |

### 4. 블록체인 기록

- 선택된 좌표가 AES-256으로 암호화됩니다
- 암호화된 데이터가 Ronin 블록체인에 기록됩니다
- 트랜잭션 해시가 발급됩니다

### 5. 결과 확인

- 게임 종료 시간에 결과가 발표됩니다
- 즉시 당첨 상품의 경우 참여 직후 바로 결과를 확인합니다

### 6. 상품 수령 (당첨 시)

- 실물 상품: 배송지 입력 → 배송
- 기프티쇼 쿠폰: 쿠폰 URL 즉시 발급
