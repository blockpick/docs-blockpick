## 상태 관리 (Riverpod)

### Auth Providers
- `authProvider`: 인증 상태
- `isAuthenticatedProvider`: 인증 여부
- `currentUserProvider`: 현재 사용자
- `authTokenProvider`: JWT 토큰

### Game Providers
- `gameProvider(id)`: 게임 상세
- `gamesByTypeProvider`: 타입별 게임
- `gameJoinProgressProvider`: 참여 진행
- `gameResultProvider(id)`: 결과
- `instantPrizeProvider(id)`: 즉시 당첨

### User Data Providers
- `walletProvider`: 지갑/잔액
- `pendingTransactionProvider`: 진행 중 TX
- `adRewardProvider`: 광고 리워드
- `notificationSettingsProvider`: 알림 설정
