# 아키텍처

블록픽의 시스템 아키텍처입니다.

## 앱 아키텍처 패턴

```
┌────────────────────────────────────────────────┐
│                Presentation Layer               │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐     │
│  │ Screens  │  │ Widgets  │  │ Providers│     │
│  │ (GoRouter│  │ (UI      │  │ (Riverpod│     │
│  │  Pages)  │  │  Comps)  │  │  State)  │     │
│  └────┬─────┘  └────┬─────┘  └────┬─────┘     │
├───────┼──────────────┼─────────────┼───────────┤
│       │         Domain Layer       │           │
│  ┌────┴────────────────────────────┴─────┐     │
│  │  Models (Freezed)                     │     │
│  │  Repositories (Interface)             │     │
│  │  Providers (Business Logic)           │     │
│  └────┬──────────────────────────────────┘     │
├───────┼────────────────────────────────────────┤
│       │           Data Layer                   │
│  ┌────┴──────────────────────────────────┐     │
│  │  Remote DataSources (GraphQL/Dio)     │     │
│  │  Local DataSources (Hive/SecureStore) │     │
│  │  Blockchain Service (web3dart)        │     │
│  └───────────────────────────────────────┘     │
└────────────────────────────────────────────────┘
```

## 웹 아키텍처 패턴

```
┌────────────────────────────────────────────────┐
│               Next.js App Router                │
│  ┌──────────────────────────────────────┐      │
│  │  Pages (app/)                         │      │
│  │  ├── page.tsx (Landing)               │      │
│  │  ├── game/[id]/page.tsx (Game Detail) │      │
│  │  └── more/[id]/page.tsx (Mini Games)  │      │
│  └──────────┬───────────────────────────┘      │
│             │                                   │
│  ┌──────────┴───────────────────────────┐      │
│  │  Components (components/)             │      │
│  │  ├── ui/ (공통 UI)                    │      │
│  │  └── game/ (게임 컴포넌트, 30종+)      │      │
│  └──────────┬───────────────────────────┘      │
│             │                                   │
│  ┌──────────┴───────────────────────────┐      │
│  │  Lib (lib/)                           │      │
│  │  ├── graphql/ (Apollo Client, Queries)│      │
│  │  ├── store/ (Zustand - auth, game)    │      │
│  │  ├── bridge/ (WebView Bridge)         │      │
│  │  └── utils/ (Canvas, Particles)       │      │
│  └──────────────────────────────────────┘      │
└────────────────────────────────────────────────┘
```
