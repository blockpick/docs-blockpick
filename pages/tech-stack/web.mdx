│   │       └── [id]/             # 30종+ 미니게임

│   │   ├── ui/                   # 공통 UI (Modal, Nav, Toast)
│   │   └── game/                 # 게임 컴포넌트
│   │       ├── GachaCanvas.tsx   # 메인 가챠 캔버스

│   │       ├── abyssal/          # 심해 게임 시스템
│   │       └── singularity/      # 우주 게임 시스템
│   └── lib/
│       ├── graphql/              # Apollo 클라이언트, 쿼리, 뮤테이션

## 상태 관리 (Zustand)

### Auth Store
- `user`, `isLoggedIn`
- `accessToken`, `refreshToken`
- `showLoginModal`, `showPhoneModal`
- `context` (app vs web 감지)
- `login()`, `logout()`, `restoreSession()`

### Game Store
- `game`, `phase` (loading → ready → result)
- `selectedBlock` \{row, col\}
- `prize`, `isWin`, `entryId`
- `guestToken` (비회원)
- `selectBlock()`, `setPrize()`, `reset()`
