# 기술 용어 사전

| 용어 | 설명 |
|------|------|
| **AES-256** | 256비트 대칭키 암호화 알고리즘 |
| **Apollo Client** | JavaScript용 GraphQL 클라이언트 |
| **BIP39** | 블록체인 지갑 니모닉 시드 문구 표준 |
| **CDN** | Content Delivery Network |
| **EVM** | Ethereum Virtual Machine |
| **Freezed** | Dart 불변 데이터 클래스 코드 생성 |
| **GoRouter** | Flutter 선언적 라우팅 |
| **GraphQL** | API 쿼리 언어 |
| **idempotencyKey** | 중복 요청 방지 고유 키 |
| **JWT** | JSON Web Token |
| **LOD** | Level of Detail |
| **Matter.js** | JavaScript 2D 물리 엔진 |
| **MediaPipe** | Google 실시간 미디어 처리 |
| **Rapier** | Rust 기반 3D 물리 엔진 |
| **Riverpod** | Flutter 반응형 상태 관리 |
| **Ronin** | EVM 호환 블록체인 |
| **Three.js** | JavaScript 3D 그래픽 (WebGL) |
| **web3dart** | Dart용 Ethereum 클라이언트 |
| **WebView** | 앱 내 웹 표시 컴포넌트 |
| **Zustand** | React 경량 상태 관리 |
