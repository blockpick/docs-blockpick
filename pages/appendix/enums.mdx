# Enum 정의

## GameType

| 값 | 설명 |
|----|------|
| `DAILY` | 매일 진행 일일 게임 |
| `SELECT` | 그리드 직접 선택 |
| `VIBE` | 분위기/테마 기반 |
| `PRIME` | 프리미엄 역경매 |
| `PARTNER` | 파트너 협업 |

## GameStatus

| 값 | 설명 |
|----|------|
| `SCHEDULED` | 예정됨 |
| `ACTIVE` | 진행 중 |
| `ENDED` | 종료됨 |
| `CANCELLED` | 취소됨 |

## EntryStatus

| 값 | 설명 |
|----|------|
| `PENDING` | 대기 중 |
| `CONFIRMED` | 확인됨 |
| `WON` | 당첨 |
| `LOST` | 낙첨 |

## UserStatus

| 값 | 설명 |
|----|------|
| `ACTIVE` | 활성 |
| `SUSPENDED` | 정지됨 |
| `WITHDRAWN` | 탈퇴됨 |

## TransactionType

| 값 | 설명 |
|----|------|
| `CHARGE` | 충전 |
| `SPEND` | 사용 |
| `REFUND` | 환불 |
| `REWARD` | 보상 |

## CurrencyType

| 값 | 설명 |
|----|------|
| `SHOPPING_CASH` | 쇼핑캐시 (유료) |
| `EVENT_POINT` | 이벤트 포인트 (무료) |

## PrizeType

| 값 | 설명 |
|----|------|
| `MAIN` | 메인 당첨 상품 |
| `INSTANT` | 즉시 당첨 (기프티쇼) |

## AuthProvider

| 값 | 설명 |
|----|------|
| `EMAIL` | 이메일 가입 |
| `GOOGLE` | Google 소셜 |
| `APPLE` | Apple 소셜 |
