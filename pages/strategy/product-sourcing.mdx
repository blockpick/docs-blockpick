## 상품 조달 채널

### 채널별 개요

| 채널 | 상품 유형 | 조달 방식 | 리드타임 | 비용 구조 |
|------|---------|---------|--------|---------|
| **Giftshow API** | 모바일 상품권, 기프티콘 | API 자동 발행 | 즉시 | 권면가 대비 5~10% 마진 |
| **제휴 쇼핑몰** | 실물 상품 (가전, 패션, 식품) | 발주 → 입고 → 배송 | 3~7일 | 판매가 대비 15~25% 마진 |
| **브랜드 직접 협업** | 브랜드 한정 상품, 체험권 | 계약 기반 무상/유상 제공 | 2~4주 | 협의 (일부 무상) |
| **드롭쉬핑 파트너** | 롱테일 실물 상품 | 당첨 후 파트너사 직접 배송 | 5~10일 | 판매가 대비 20~30% 마진 |

### 채널별 세부 운영 방식

#### Giftshow API (기프티콘 / 모바일 상품권)

- API 연동을 통해 당첨 즉시 쿠폰 자동 발행
- 대상 상품: 스타벅스, 배달앱 쿠폰, 편의점 상품권, OTT 이용권, 게임 아이템 등
- 재고 부족 시 자동 대체 쿠폰 발행 설정 (동급 브랜드로 대체)
- 쿠폰 유효기간 관리: 발행 후 30~90일 이내 사용 권장 안내 포함

<Callout type="info">
  기프티콘 상품은 **당첨 즉시 자동 발행**되므로 수동 배송 리드타임이 없습니다. DAILY·VIBE 게임의 기본 상품으로 가장 적합합니다.
</Callout>

#### 제휴 쇼핑몰 (실물 상품)

- 당첨 확정 후 D+1 발주 → D+7 이내 배송 완료 목표
- 협력 쇼핑몰: 11번가, 쿠팡 비즈니스, 위메프 등 (협의 중)
- 재고 확인 후 게임 오픈: 최소 게임 개설 수량 × 2배 재고 확보 원칙
- 실물 상품 교환·반품: CS팀이 파트너몰과 직접 처리

#### 브랜드 직접 협업

- 파트너십 계약 기반: 상품 무상 제공 + 블록픽 내 브랜드 노출 교환
- 대상: 중소 규모 뷰티, 식품, 라이프스타일 브랜드
- 협업 과정: 제안서 발송 → 미팅 → 상품 샘플 검수 → 계약 → 게임 오픈
- 계약 조건: 상품 무상 제공, 브랜드 로고 노출, 당첨 후기 SNS 공유 의무화

#### 드롭쉬핑 파트너

- 블록픽이 직접 재고를 보유하지 않고 파트너사가 당첨자에게 직배송
- 적합 상품: 고가 가전, 가구, 대형 상품
- 파트너사 SLA: 발주 후 5영업일 이내 배송 시작
- 배송 추적 번호 블록픽 앱 내 제공 의무

## 상품 카테고리 전략

### 카테고리별 비중 (목표 포트폴리오)

| 카테고리 | 비중 목표 | 주력 상품 예시 | 게임 유형 |
|---------|---------|------------|---------|
| **전자기기 / 가전** | 15% | AirPods, 갤럭시 워치, 무선청소기 | PRIME, DAILY |
| **뷰티 / 헬스** | 20% | 화장품 세트, 향수, 마스크팩 | VIBE, DAILY |
| **패션 / 잡화** | 15% | 신발, 가방, 액세서리 | VIBE, PRIME |
| **식품 / 음료** | 20% | 카페 쿠폰, 밀키트, 건강식품 | DAILY, VIBE |
| **체험 / 여행** | 10% | 레스토랑 식사권, 호텔 숙박권 | PRIME |
| **디지털 콘텐츠** | 10% | OTT 구독권, 앱 구독, 게임 아이템 | DAILY |
| **생활 / 홈** | 10% | 쿠팡 상품권, 생활용품 쿠폰 | DAILY, VIBE |

<Callout type="warning">
  특정 카테고리가 전체의 30%를 초과하면 유저 이탈 위험이 높아집니다. 분기별 카테고리 비중을 검토하고 균형을 유지합니다.
</Callout>

### 가격대별 비중 및 게임 매핑

| 가격대 | 비중 목표 | 주력 게임 유형 | 참여비 기준 |
|--------|---------|------------|---------|
| **1만원 미만** | 30% | DAILY 무료 참여권 활용 | 0P (무료) ~ 10P |
| **1~5만원** | 35% | VIBE (소원 10P) | 10P |
| **5~20만원** | 25% | PRIME 역경매 | 100P~500P |
| **20만원 이상** | 10% | PRIME 프리미엄 | 1,000P+ |

### DAILY vs WISH vs PRIME 상품 배분 기준

| 기준 | DAILY | WISH (VIBE) | PRIME |
|------|-------|------------|-------|
| **가격대** | 1만원 이하 위주 | 1~10만원 | 5만원 이상 |
| **상품 희소성** | 범용 (대중적) | 원하는 것 (감성적) | 고가 한정 |
| **참여 장벽** | 없음 (무료) | 낮음 (10P) | 중간~높음 |
| **배분 비중** | 40% | 40% | 20% |
| **주력 카테고리** | 식품, 디지털 콘텐츠 | 뷰티, 패션 | 가전, 체험, 명품 |

## 상품 큐레이션 원칙

### 타겟 유저층별 인기 상품

| 유저 세그먼트 | 추정 연령 | 선호 카테고리 | 참여 게임 |
|-------------|--------|------------|---------|
| **트렌드 세터** | 20대 초반 | 뷰티, 패션, 콘텐츠 구독 | VIBE, DAILY |
| **실속 추구형** | 20~30대 | 식품 쿠폰, 상품권 | DAILY |
| **경험 중시형** | 30대 | 체험권, 여행, 다이닝 | PRIME |
| **테크 얼리어답터** | 20~40대 남성 | 가전, 게임, 디지털 기기 | PRIME |
| **선물 목적형** | 30~40대 | 명품, 선물세트, 백화점 쿠폰 | PRIME |

### 시즌별 상품 로테이션 원칙

- **1월~2월**: 신년 / 발렌타인 → 전자기기, 뷰티 비중 확대
- **3월~5월**: 봄 / 가정의 달 → 체험권, 패션, 키즈 상품 확대
- **6월~8월**: 여름 성수기 → 여행·숙박, 뷰티, 가전(냉방) 비중 최대화
- **9월~10월**: 추석 / 핼러윈 → 선물세트, 식품, 코스튬 집중
- **11월~12월**: 블프 / 연말 → 전자기기, 명품, 선물 카테고리 피크

### 화제성 / 트렌드 상품 선정 기준

| 기준 | 판단 방법 | 갱신 주기 |
|------|---------|--------|
| SNS 언급량 | 인스타그램·틱톡 해시태그 조회 수 | 주 1회 |
| 네이버 검색 트렌드 | 검색어 급상승 키워드 | 주 1회 |
| 유튜브 콘텐츠 노출 | 인플루언서 최근 리뷰 상품 | 주 2회 |
| 커머스 인기 순위 | 쿠팡·무신사·올리브영 베스트 | 실시간 |
| 유저 소원 등록 상품 | 블록픽 내 소원 등록 수 TOP 20 | 매일 |

<Callout type="info">
  **블록픽 소원 등록 데이터**는 가장 신뢰도 높은 상품 수요 지표입니다. 소원 상위 20개 상품을 다음 달 WISH/PRIME 게임 상품 후보로 우선 검토합니다.
</Callout>

### 가격 임팩트 설계 원칙

참여비 10P 기준, 상품가 10만원의 경우 유저 입장에서 "100원으로 10만원 상품에 도전"이라는 임팩트를 제공합니다.

| 참여비 | 권장 상품가 범위 | 배수 효과 | 예시 상품 |
|-------|-------------|--------|---------|
| 0P (무료) | 1,000~10,000원 | 무한 | 커피 쿠폰, 편의점 상품권 |
| 10P (VIBE) | 10,000~50,000원 | 1,000~5,000배 | 뷰티 세트, 패션 아이템 |
| 100P (PRIME) | 50,000~200,000원 | 500~2,000배 | 무선이어폰, 향수, 체험권 |
| 1,000P (PRIME+) | 500,000원 이상 | 500배 이상 | 명품, 여행권, 고가 가전 |

## 상품 관리 프로세스

### 재고 관리 원칙

| 상품 유형 | 재고 기준 | 관리 방식 |
|---------|---------|---------|
| 모바일 쿠폰 (Giftshow) | API 잔여 재고 실시간 조회 | 10% 미만 시 자동 경고 발생 |
| 실물 상품 | 게임 개설 수량 × 1.5배 사전 확보 | 발주 후 입고 확인 필수 |
| 체험권 / 예약권 | 날짜별 사용 가능 수량 확인 | 기간 만료 전 소진 원칙 |
| 파트너사 공급 상품 | 파트너사 재고 확인 후 게임 오픈 | 품절 시 즉시 게임 마감 처리 |

### 품절 / 배송 지연 대응

| 상황 | 발생 기준 | 대응 방법 | 유저 공지 |
|------|---------|---------|---------|
| 쿠폰 재고 소진 | 게임 마감 전 재고 0 도달 | Giftshow 대체 쿠폰 자동 발행 | 자동 알림 (대체 상품 안내) |
| 실물 재고 부족 | 당첨자 수 > 보유 재고 | 동급 상품으로 대체 또는 동의 후 환불 | CS 직접 연락 |
| 배송 3일 이상 지연 | 발송 후 추적 정지 3일 | 파트너사 확인 → CS 안내 | 앱 푸시 + CS 연락 |
| 상품 하자 발견 | 수령 후 유저 신고 | 교환 또는 등가 포인트 환불 | CS 처리 후 결과 안내 |

<Callout type="warning">
  당첨 후 상품을 받지 못한 유저는 **서비스 신뢰도에 치명적 영향**을 미칩니다. CS 처리 목표: 당첨 후 7일 이내 100% 상품 수령 완료.
</Callout>

### 상품 교환 / 반품 정책

| 정책 | 내용 |
|------|------|
| 모바일 쿠폰 교환 | 미사용 쿠폰에 한해 동일 금액 쿠폰 재발행 (유효기간 내) |
| 실물 상품 반품 | 수령 후 7일 이내 미개봉 상품에 한해 반품 접수 |

| 환불 원칙 | 현금 환불 없음, 포인트 또는 동급 상품 교환으로 처리 |

## 상품 비용 최적화

### 쿠폰 대량 구매 할인 구조

| 구매 규모 | 할인율 | 적용 기준 |
|---------|-------|---------|
| 월 100만원 미만 | 기본가 | 없음 |
| 월 100~500만원 | 3% 할인 | Giftshow 협의 |
| 월 500만원 이상 | 5~8% 할인 | 개별 계약 |
| 연간 선구매 | 10% 할인 | 연간 예산 확정 기준 |

### 파트너 상품 무상 제공 모델

| 모델 | 블록픽 제공 가치 | 파트너사 제공 가치 | 적합 대상 |
|------|--------------|---------------|---------|
| 브랜드 노출형 | 게임 배너 + 결과 화면 노출 | 상품 무상 제공 | 신제품 런칭 브랜드 |
| 데이터 공유형 | 참여 유저 선호도 데이터 | 상품 무상 제공 | 시장조사 필요 브랜드 |
| 공동 캠페인형 | SNS 콘텐츠 공동 제작 | 상품 무상 제공 + 홍보비 분담 | 중소 뷰티·식품 |
| 독점 소싱형 | 블록픽 독점 판매권 | 가격 할인 제공 | 한정판 상품 보유 브랜드 |

### 당첨 상품 원가율 관리

<Callout type="info">
  **목표 원가율**: 당첨 상품 원가 합계 / 해당 게임 전체 참여비 합계 = **60~80%**
  이 범위를 유지해야 서비스 지속 가능성과 유저 만족도를 동시에 확보할 수 있습니다.
</Callout>

```
원가율 계산 예시:
- 상품: 5만원 짜리 뷰티 세트
- VIBE 게임 참여비: 10P (유료 충전 시 100원 상당)
- 손익분기 참여자 수: 5만원 / 100원 = 500명
- 목표 원가율 70% 기준: 500명 / 0.7 = 약 715명 참여 시 수익 발생
- 따라서 이 게임은 참여자 715명 이상 모집 시 운영 수익 확보
```

| 원가율 구간 | 판단 | 조치 |
|-----------|------|------|
| 60% 미만 | 상품 임팩트 부족 위험 | 더 높은 가격대 상품으로 교체 |
| 60~80% | 정상 운영 범위 | 유지 |
| 80~90% | 수익성 경계 | 참여비 인상 또는 참여 인원 제한 검토 |
| 90% 초과 | 손익 위험 | 게임 구조 재설계 필요 |

### 카테고리별 원가율 목표

| 카테고리 | 원가율 목표 | 비고 |
|---------|---------|------|
| 모바일 쿠폰 | 70~80% | 마진 낮음, 대신 조달 비용 최소화 |
| 뷰티 / 식품 | 65~75% | 대량 구매 할인 가능 |
| 전자기기 | 60~70% | 고가 상품으로 임팩트 극대화 |
| 체험권 | 55~70% | 파트너사 무상 제공 시 원가율 낮아짐 |
| 파트너 무상 상품 | 0% (원가 없음) | 브랜드 가치 교환 |
