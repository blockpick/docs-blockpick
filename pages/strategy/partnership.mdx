# 파트너십 & B2B 수익 모델

블록픽의 B2B 수익은 **브랜드가 직접 게임을 개설하는 '업체 소원' 광고 모델**을 중심으로, 브랜드 콜라보 게임, 드롭쉬핑 마진, 스폰서 노출 패키지로 구성됩니다. B2B 수익은 유저 수에 비례하지 않고 파트너 계약에 따라 예측 가능한 고정 수익을 제공한다는 점에서 전략적 가치가 큽니다.

<Callout type="info">
**핵심 가치 제안:** 블록픽 파트너사는 광고비를 '소진'하는 것이 아니라, 실제 게임 참여자 → 브랜드 체험 → 구매 전환으로 이어지는 **퍼포먼스형 브랜드 경험**을 구매합니다.
</Callout>

---

## 1. 업체 소원 광고 모델

### 개요

업체(브랜드/셀러)가 자사 상품을 당첨품으로 제공하는 소원 게임을 직접 개설합니다. 유저들이 포인트를 소비하여 참여하고, 블록픽 블록체인이 공정 추첨을 집행하며, 당첨자에게 브랜드 상품이 지급됩니다.

```
브랜드 광고비 = 게임 개설비 + 노출 패키지비 + (옵션) 프리미엄 배치비
```

### 광고 노출 위치

| 노출 위치 | 설명 | 노출 가능 유저 |
|----------|------|--------------|
| 메인 피드 상단 | 홈 화면 최상단 카드 | 전체 DAU |
| 소원 탭 인기 | '소원' 탭 내 추천 게임 | 소원 탭 방문자 |
| 추천 배너 | 게임 상세 하단 추천 슬롯 | 게임 참여 중 유저 |
| 푸시 알림 | 게임 시작 / 마감 임박 알림 | 동의 유저 전체 |
| 로딩 인터스티셜 | 앱 로딩 중 브랜드 노출 | 전체 DAU |

### 과금 모델

| 과금 방식 | 설명 | 적합 목적 |
|----------|------|---------|
| **고정 패키지** | 기간/노출 위치 기준 고정 요금 | 브랜드 인지도 캠페인 |
| **CPM** | 1,000회 노출당 과금 | 대규모 인지도 확산 |
| **CPC** | 게임 참여 클릭당 과금 | 참여 유도 중심 |
| **CPA** | 게임 완료(참여) 건당 과금 | 퍼포먼스 최적화 |
| **Rev-Share** | 당첨 상품 판매 연동 수익 배분 | 드롭쉬핑 파트너 |

### 가격 체계

| 패키지명 | 기간 | 포함 항목 | 가격 |
|---------|------|---------|------|
| **스타터** | 7일 | 소원 1개 개설 + 기본 노출 | ₩300,000 |
| **베이직** | 14일 | 소원 1개 + 피드 노출 + 성과 리포트 | ₩700,000 |
| **스탠다드** | 30일 | 소원 2개 + 추천 배치 + 푸시 + 리포트 | ₩1,500,000 |
| **프리미엄** | 30일 | 소원 5개 + 전 위치 노출 + 전담 매니저 | ₩4,000,000 |
| **엔터프라이즈** | 협의 | 맞춤형 캠페인 + API 연동 + 풀 리포트 | 협의 |

<Callout type="info">
**파트너 ROI 시뮬레이션 (스탠다드 기준):**
- 광고비: ₩1,500,000
- 예상 게임 참여자: 5,000명
- 브랜드 노출 CPM: ₩300 (시장 평균 대비 80% 저렴)
- 당첨자 브랜드 경험 → 주변 공유 효과 포함 시 실질 CPM ₩150 수준
</Callout>

## 2. 브랜드 콜라보 게임

### 개요

브랜드가 자사 아이덴티티로 블록픽 게임을 완전히 커스터마이징하는 프리미엄 B2B 상품입니다. 단순 소원 개설을 넘어 브랜드 전용 게임 UI, 미니게임 테마, 캐릭터 통합이 포함됩니다.

### 콜라보 게임 유형

| 유형 | 설명 | 예시 |
|------|------|------|
| **브랜드 DAILY** | 브랜드 로고/색상으로 커스텀된 데일리 추첨 | 스타벅스 DAILY |
| **브랜드 미니게임** | 브랜드 캐릭터 테마 미니게임 1종 전용 개발 | 라인프렌즈 슬라이드 퍼즐 |
| **시즌 콜라보** | 특정 기간 전체 테마 브랜딩 (홀리데이 등) | 크리스마스 × 편의점 브랜드 |
| **런칭 캠페인** | 신제품 출시에 맞춘 PRIME 역경매 이벤트 | 신상 스니커즈 역경매 |

### 커스터마이징 항목

| 항목 | 스탠다드 | 프리미엄 | 엔터프라이즈 |
|------|---------|---------|------------|
| 게임 배경/컬러 | ✅ | ✅ | ✅ |
| 브랜드 로고 삽입 | ✅ | ✅ | ✅ |
| 캐릭터/IP 통합 | ❌ | ✅ | ✅ |
| 전용 미니게임 개발 | ❌ | ❌ | ✅ |
| 앱 스플래시 브랜딩 | ❌ | ✅ | ✅ |
| 전용 URL 딥링크 | ❌ | ✅ | ✅ |
| API 연동 (CRM 등) | ❌ | ❌ | ✅ |

### 브랜드 콜라보 성과 리포트 제공 항목

- 게임 참여자 수 / 완료율
- 유저 연령대 / 성별 분포 (익명 집계)
- 일별 노출 수 / CTR
- 당첨자 소셜 공유 건수 (바이럴 지수)
- 기간 내 앱 신규 가입 기여 수 (라스트클릭 기준)

## 3. 드롭쉬핑 모델

### 개요

블록픽이 파트너사 상품을 당첨 상품으로 직접 소싱하거나, 파트너사 재고를 위탁 판매하여 중간 마진을 수익으로 가져가는 구조입니다.

### 드롭쉬핑 흐름

```
[파트너사 상품 등록] → [블록픽 게임 당첨 상품으로 노출]
         → [유저 당첨] → [블록픽 → 파트너사 발주]
                → [파트너사 직배송] → [마진 정산]
```

### 수익 구조

| 구분 | 내용 | 마진율 |
|------|------|--------|
| **위탁 드롭쉬핑** | 파트너사가 직접 배송, 블록픽은 플랫폼 수수료 수취 | 10% ~ 20% |
| **매입 드롭쉬핑** | 블록픽이 사전 매입 후 게임 운영, 차익 수취 | 20% ~ 40% |
| **쿠폰/기프티콘** | 온라인 상품권 즉시 지급, 원가 대비 수익 | 5% ~ 15% |
| **해외 직소싱** | 해외 공장 직접 소싱, 고마진 실물 상품 | 30% ~ 60% |

### 파트너사 커미션 구조

| 파트너 등급 | 월 거래액 | 블록픽 수수료 | 파트너 정산율 |
|-----------|---------|------------|------------|
| 브론즈 | ~₩500만 | 20% | 80% |
| 실버 | ₩500만 ~ ₩2,000만 | 15% | 85% |
| 골드 | ₩2,000만 ~ ₩5,000만 | 12% | 88% |
| 플래티넘 | ₩5,000만+ | 10% | 90% |

<Callout type="warning">
**드롭쉬핑 리스크:** 배송 지연, 불량 상품, 재고 부족은 블록픽 유저 신뢰도에 직접 영향을 미칩니다. 파트너사 온보딩 시 품질 기준 및 SLA(Service Level Agreement) 계약이 필수입니다.
</Callout>

## 4. B2B 영업 전략

### 4-1. 타겟 업종 / 업체

| 우선순위 | 업종 | 이유 |
|---------|------|------|
| 🥇 1순위 | 식품/음료 (카페, 편의점 브랜드) | 소비재, 빠른 의사결정, 넓은 인지도 |
| 🥇 1순위 | 뷰티/화장품 | 20~30대 여성 타겟 겹침, 샘플 제공 용이 |
| 🥈 2순위 | 패션/스니커즈 | 한정판 심리 + PRIME 역경매 시너지 |
| 🥈 2순위 | 배달/O2O 서비스 | 쿠폰 기반 협업, 즉시 지급 가능 |
| 🥉 3순위 | 가전/IT 기기 | 고가 상품으로 PRIME 참여 유도 |
| 🥉 3순위 | 게임/엔터테인먼트 | 게임 아이템/시즌권 협업 |

### 4-2. 영업 퍼널

```
[인바운드 / 아웃바운드 발굴]
        ↓
[파트너 제안서 + ROI 시뮬레이션 제공]
        ↓
[무료 파일럿 캠페인 (7일, 스타터 패키지)]
        ↓
[성과 리포트 공유 → 재계약 제안]
        ↓
[월 정기 계약 또는 연간 패키지 전환]
```

| 단계 | 기간 | 목표 전환율 |
|------|------|-----------|
| 발굴 → 미팅 | 1~2주 | 30% |
| 미팅 → 파일럿 | 1~2주 | 50% |
| 파일럿 → 계약 | 2~4주 | 60% |
| 계약 → 재계약 | 1개월 후 | 70% |

### 4-3. 파일럿 프로그램

신규 파트너사의 리스크를 낮추고 블록픽의 광고 효과를 증명하는 온보딩 프로그램입니다.

| 항목 | 내용 |
|------|------|
| 기간 | 7일 |
| 비용 | 무료 (상품 원가만 파트너사 부담) |
| 포함 내용 | 소원 게임 1개 개설 + 기본 노출 + 성과 리포트 |
| 목표 | 게임 참여자 500명 이상, CTR 5% 이상 |
| 종료 후 | 성과 리포트 + 유료 패키지 제안 |

<Callout type="info">
**파일럿 성과 기준:** 500명 참여 / 5% CTR / 당첨자 소셜 공유율 30% 이상을 달성하면 ROI 양호로 판단하여 유료 전환을 제안합니다.
</Callout>

## 5. B2B 수익 성장 로드맵

| Phase | 시기 | 파트너 수 | 월 B2B 매출 목표 | 주요 활동 |
|-------|------|---------|----------------|---------|
| Phase 1 | 출시 ~ 6개월 | 1 ~ 5개 | ₩0 ~ ₩500만 | 파일럿 케이스 구축, 레퍼런스 확보 |
| Phase 2 | 7 ~ 18개월 | 5 ~ 30개 | ₩500만 ~ ₩3,000만 | 영업팀 구성, 표준 패키지 출시 |
| Phase 3 | 19개월+ | 30개+ | ₩3,000만+ | 셀프서브 광고 플랫폼, 대형 브랜드 계약 |

### 셀프서브 광고 플랫폼 (Phase 3 목표)

MAU 10만 이상 달성 시, 파트너사가 영업 담당자 없이 직접 캠페인을 집행할 수 있는 셀프서브 대시보드를 구축합니다.

| 기능 | 내용 |
|------|------|
| 캠페인 생성 | 상품 등록, 게임 유형 선택, 예산 설정 |
| 타겟팅 | 연령대, 성별, 관심사, 지역 필터 |
| 실시간 대시보드 | 노출, 참여, CTR, 전환 실시간 확인 |
| 자동 입찰 | CPM/CPC 자동 최적화 입찰 |
| 정산 | 월별 자동 정산 및 세금계산서 발행 |

## 6. 파트너사 성과 지표 (KPI)

파트너사에 제공하는 성과 리포트의 핵심 지표입니다.

| KPI | 정의 | 목표 기준값 |
|-----|------|-----------|
| **게임 참여율** | 노출 대비 참여 완료 | 5% 이상 |
| **CPE** | Cost Per Engagement (참여당 비용) | ₩500 이하 |
| **브랜드 리콜률** | 참여 후 브랜드 인지 (설문) | 60% 이상 |
| **소셜 공유율** | 당첨자 중 SNS 공유 비율 | 30% 이상 |
| **재참여율** | 동일 브랜드 게임 재참여 | 20% 이상 |
| **전환율** | 게임 참여 → 실제 구매 (연동 시) | 2% 이상 |
