## 이벤트 템플릿

### 정기 이벤트 실행 템플릿

```
이벤트명: 주간 DAILY 챌린지
목적: 주초 재방문율 상승 (월요일 DAU 목표치 달성)
기간: 매주 월~수 (3일)
참여 조건: 해당 주 DAILY 게임 1회 이상 참여
보상: 당첨자 외 참여자 전원 10P 위로 포인트
KPI: 월요일 DAU 목표 대비 달성률, 게임 참여 전환율
담당: 기획 1명, CS 대기, 개발 불필요
```

### 시즌 이벤트 실행 템플릿

```
이벤트명: [시즌명] 이벤트
목적: 시즌 감성 연계 DAU 상승 + 충전 전환
기간: 시즌 피크 전후 7~14일
상품: 시즌 테마 카테고리 집중 (3~5종)
특별 UI: 홈 배너 교체, 로딩 화면 시즌 테마
보상 구조:
  - 참여 포인트 지급: 20~50P
  - 특별 당첨 확률 UP 이벤트 옵션
  - SNS 공유 시 추가 보상 10P
KPI: DAU 평시 대비 +20%, 충전 전환율 +5%p, 소원 공유 횟수
리드타임: 최소 3주 전 기획 확정
```

### 돌발 이벤트 실행 템플릿

```
이벤트명: 번개 핫딜
목적: 즉각적 트래픽 유입 및 참여율 회복
기간: 12~24시간 (상황에 따라 연장 가능)
공지: 푸시 알림 즉시 발송 + 홈 상단 배너
상품: 단일 인기 상품 1종 집중
KPI: 이벤트 기간 DAU, 게임 참여 전환율
담당: 기획 1명 (어드민 직접 설정)
개발 의존: 없음
```

## 이벤트 효과 측정

### 핵심 측정 지표

| 지표 | 설명 | 측정 도구 | 목표 기준 |
|------|------|---------|---------|
| DAU 증감율 | 이벤트 기간 DAU vs 직전 동기간 | 내부 대시보드 | +15% 이상 |
| 게임 참여 전환율 | 이벤트 인지 → 게임 참여 | Amplitude Funnel | 40% 이상 |
| 충전 전환율 | 이벤트 기간 유료 충전 비율 | 결제 시스템 | +5%p 이상 |
| 신규 유저 비율 | 이벤트 유입 중 신규 가입 비율 | 내부 대시보드 | 20% 이상 |
| 소원 공유 횟수 | 이벤트 기간 SNS 공유 수 | 딥링크 추적 | 평시 대비 +50% |
| 당첨 후기 등록율 | 당첨자 중 후기 작성 비율 | 내부 대시보드 | 30% 이상 |

### 이벤트 ROI 계산식

```
이벤트 ROI (%) = (이벤트 기간 증분 매출 - 이벤트 비용) / 이벤트 비용 × 100

증분 매출 = 이벤트 기간 유료 충전 총액 - 직전 동기간 유료 충전 총액
이벤트 비용 = 상품 원가 + 무료 포인트 지급 환산액 + 운영 인건비

목표: 시즌 이벤트 ROI 150% 이상 / 정기 이벤트 ROI 200% 이상
```
