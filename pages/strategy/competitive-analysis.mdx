## 1. 시장 정의

### 1-1. 핵심 시장: 국내 게이미피케이션 커머스

블록픽이 직접 경쟁하는 시장은 **게임 요소(참여·추첨·미션)와 커머스(상품 획득)를 결합한 앱 서비스** 영역입니다.

| 시장 구분 | 규모 (2024 추정) | 연간 성장률 | 핵심 동인 |
|---|---|---|---|
| 게이미피케이션 커머스 | 약 3,000억 원 | 25~30% | MZ세대 엔터테인먼트 구매, 가성비 추구 |
| 리워드/포인트 앱 | 약 8,000억 원 | 15% | 인플레이션 대응, 절약 문화 |
| 래플/한정판 커머스 | 약 5,000억 원 | 40% | 한정판 문화, 리셀 시장 성장 |

### 1-2. 인접 시장

**리워드 앱 (캐시워크, 토스)**
- 걷기·광고 시청 → 포인트 → 현금성 보상
- 공통점: 무료 포인트 획득 → 유료 서비스 연결 루프
- 차이점: 블록픽은 포인트를 추첨 참여에 소비 (소비 맥락 = 기대감)

**럭키백/래플 (나이키 SNKRS, 크림)**
- 한정판 상품 + 추첨 → 공정성 요구 높음
- 공통점: 추첨 결과의 공정성이 신뢰 핵심
- 차이점: 블록픽은 블록체인으로 공정성을 온체인 증명

**소셜 커머스 (공동구매, 올웨이즈)**
- 지인 초대 → 할인/무료 상품 획득
- 공통점: 소셜 바이럴이 핵심 성장 엔진
- 차이점: 블록픽은 소원(위시) 기반 정서적 공유 유도

## 2. 경쟁사 맵

### 2-1. 직접 경쟁: 추첨/게임 커머스

<Callout type="warning">
  직접 경쟁사들은 **추첨 투명성 부재**로 유저 신뢰 이탈을 겪고 있습니다. 이 공백이 블록픽의 핵심 진입 기회입니다.
</Callout>

| 서비스 | 핵심 메커니즘 | 월 활성 유저 | 수익 모델 | 취약점 |
|---|---|---|---|---|
| 원더클릭 | 클릭 기반 즉석 추첨 | ~50만 MAU | 참여비 + 광고 | 투명성 논란, 사행성 이미지 |
| 럭키팡 | 포인트 소모 뽑기 | ~30만 MAU | 포인트 판매 | 낮은 재참여율, 게임 단조 |
| 뽑기박스 | 박스형 랜덤 상품 | ~20만 MAU | 박스 판매 | 상품 품질 불만, 바이럴 없음 |
| 캐치포켓 | 미션 기반 추첨 | ~15만 MAU | 광고 + 충전 | 미션 반복성 피로 |

### 2-2. 간접 경쟁: 리워드·소셜 커머스

| 서비스 | 월 활성 유저 | 게이미피케이션 요소 | 블록픽과의 관계 |
|---|---|---|---|
| 캐시워크 | ~1,000만 MAU | 걷기 → 캐시 | 포인트 획득 행동 습관 형성 (교육 효과) |
| 토스 | ~2,000만 MAU | 미니게임, 행운퀴즈 | 게임 참여 습관 있는 유저 풀 보유 |
| 올웨이즈 | ~300만 MAU | 물뿌리기, 공동구매 | 소셜 바이럴 커머스 선례 |
| 포켓CU | ~500만 MAU | 스탬프, 이벤트 | 오프라인 연계 게이미피케이션 |

### 2-3. 잠재 경쟁: 대기업 게이미피케이션 확장

<Callout type="error">
  **중장기 최대 위협**: 쿠팡·네이버가 기존 트래픽에 추첨 기능을 붙이면 즉시 규모 경쟁이 불가해집니다. 블록픽은 **공정성 증명**과 **커뮤니티 밀도** 로 차별화해야 합니다.
</Callout>

| 기업 | 잠재 진입 방식 | 진입 가능성 | 대응 전략 |
|---|---|---|---|
| 쿠팡 | 로켓배송 + 래플 결합 | 중 (2~3년 내) | 커뮤니티·감성 차별화 |
| 네이버 | 쇼핑라이브 + 추첨 | 중~고 | 블록체인 공정성 선점 |
| 카카오 | 선물하기 + 추첨 | 중 | 소원 공유 = 카카오와 연동 전략 |
| 배민 | 음식 주문 + 게임 | 낮음 | 비식품 카테고리 집중 |

## 3. 경쟁 우위 분석 (SWOT)

### 3-1. SWOT 매트릭스

| | 긍정 요인 | 부정 요인 |
|---|---|---|
| **내부 요인** | **Strength** | **Weakness** |
| | 블록체인 온체인 공정성 증명 | 신규 서비스 브랜드 인지도 부재 |
| | 소원 기반 감성적 바이럴 루프 | 블록체인/크립토 UX 복잡성 우려 |
| | 미니게임 30종+ 체류 시간 확보 | 초기 상품 파트너사 수 제한 |
| | DAILY·WISH·PRIME 3종 게임 다양성 | 포인트 경제 밸런스 조정 난이도 |
| **외부 요인** | **Opportunity** | **Threat** |
| | 추첨 조작 의심 → 공정성 수요 급증 | 사행성 규제 강화 (게임산업법 개정) |
| | MZ세대 게이미피케이션 소비 선호 | 대기업(쿠팡·네이버) 유사 기능 출시 |
| | 블록체인 대중화 → 신뢰 인프라 수용 | 유저 포인트 피로도 및 이탈 |
| | 래플 시장 고성장 (연 40%) | 암호화폐 시장 변동성 연동 이미지 |

### 3-2. SO 전략 (강점 × 기회)

```
블록체인 공정성 증명 × 공정성 수요 급증
→ "블록해시가 증명하는 추첨" 캠페인으로 시장 선점
→ Ronin 네트워크 트랜잭션을 유저에게 직접 노출 (투명성 마케팅)

소원 바이럴 루프 × MZ 게이미피케이션 선호
→ 소원 = 감성 콘텐츠 → 인스타·X 자연 확산 유도
→ 당첨 순간 영상 바이럴 ("이거 진짜야?") 콘텐츠 전략
```

### 3-3. ST 전략 (강점 × 위협)

```
블록체인 공정성 × 규제 리스크
→ 사전 법무 검토, "상품 교환권" 방식으로 사행성 회피
→ 블록체인 공정성 = 규제 대응의 근거 자료로 활용

소원 커뮤니티 × 대기업 진입
→ 대기업이 모방 불가한 소원-감성-커뮤니티 밀도 구축
→ 초기 파워 유저 락인 (뱃지, 랭킹, 누적 포인트)
```

## 4. 포지셔닝 전략

### 4-1. 포지셔닝 맵

```
                높은 공정성/투명성
                        ↑
                        │
                   [블록픽] ← 목표 포지션
                        │
   고비용 ──────────────┼────────────── 저비용/무료
  (래플·한정판)          │              (리워드 앱)
                        │
         [원더클릭] [럭키팡]
                        │
                        ↓
                낮은 공정성/투명성
```

**블록픽의 목표 포지션**: 저비용(10원~) + 높은 공정성 증명 = "누구나 도전하는 공정한 추첨"

### 4-2. 핵심 차별화 3가지

| # | 차별화 요소 | 경쟁사 대비 | 유저 가치 |
|---|---|---|---|
| 1 | **블록체인 공정성 증명** | 경쟁사 전부 불투명 | "조작 없다"는 신뢰 |
| 2 | **소원 기반 감성 바이럴** | 단순 포인트 적립 유도 | "나도 모르게 공유하게 됨" |
| 3 | **10원~의 극소액 참여** | 수백~수천 원 단위 | "잃어도 괜찮은 도전" |

### 4-3. 메시지 프레임워크

<Callout type="info">
  모든 마케팅 메시지는 이 3가지 레이어를 기반으로 작성합니다.
</Callout>

**핵심 슬로건**
```
"10원으로 도전, 블록체인이 보장"
```

**타겟별 메시지**

| 타겟 | 핵심 메시지 | 채널 |
|---|---|---|
| 20대 여성 (소원 유저) | "이거 진짜 당첨될 수도 있어 — 소원 올려봐" | 인스타, 카카오 |
| 25~35세 남성 (PRIME 유저) | "역경매로 시세보다 싸게 — 블록체인이 공증" | X(트위터), 유튜브 |
| 가성비 추구 직장인 | "포인트 충전 없어도 미니게임으로 공짜 참여" | 네이버 카페, 커뮤니티 |
| 얼리어답터·크립토 관심 | "Ronin 블록 해시, 직접 검증해보세요" | Discord, X |

### 5-3. 벤치마크 인사이트

<Callout type="warning">
  **핵심 발견**: 직접 경쟁사들은 재참여 유인이 이벤트에 의존해 LTV가 낮습니다. 블록픽의 **미니게임 30종 + 소원 루프**는 유기적 재참여를 만드는 구조적 차별점입니다.
</Callout>

1. **공정성 공백**: 추첨 기반 경쟁사 중 블록체인 검증을 제공하는 서비스 없음 → 즉각 선점 가능
2. **감성 vs 기능**: 올웨이즈 성공 요인 = "물주기"라는 감성 행동 → 블록픽 "소원"도 동일 심리 자극
3. **체류 시간 부재**: 경쟁사 대부분 1일 1~2분 체류 → 미니게임으로 10분+ 체류 목표
4. **B2B 공백**: 브랜드가 직접 게임을 열 수 있는 플랫폼 없음 → 블록픽 파트너 소원이 차별점
