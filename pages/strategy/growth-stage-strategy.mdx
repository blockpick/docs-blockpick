## 0. 성장 전략 개요

블록픽의 성장은 **4단계 퍼널**을 기반으로 설계합니다.

```
Pre-Launch        Phase 1          Phase 2           Phase 3
(런칭 전)         (0~3개월)        (3~12개월)         (12~24개월)
   │                │                │                  │
인지도 사전 확보  → 오가닉 바이럴  → 퍼포먼스 스케일 → 브랜드 + 글로벌
MAU 0 → 사전등록  MAU 0 → 1만     MAU 1만 → 5만      MAU 5만 → 50만
```

**North Star Metric**: 월간 추첨 참여 횟수 (게임 참여 = 핵심 가치 전달)

## 2. Phase 1: Launch (0~3개월, MAU 1만 목표)

### 2-1. 핵심 전략: 오가닉 바이럴 중심

**바이럴 루프 가속 장치**

| 장치 | 메커니즘 | 기대 효과 |
|---|---|---|

| 당첨 인증 카드 | 당첨 시 공유 최적화된 이미지 자동 생성 | 당첨 바이럴 |
| 추천 코드 | 코드로 유입된 친구 첫 충전 시 추천인 포인트 | 입소문 유도 |
| DAILY 무료 참여 | 미니게임 포인트로 매일 1회 무료 | 재방문 습관 형성 |

### 2-2. 콘텐츠 마케팅

**당첨 후기 바이럴 콘텐츠 전략**

```
당첨 발생
    ↓
앱 내 당첨 영상 자동 생성 (감동 연출 + 블록 해시 표시)
    ↓
"블록체인이 골랐어요" 캡션 + 앱 링크 자동 첨부
    ↓
인스타 스토리 / 카카오 공유 1클릭
    ↓
"나도 해봤는데 진짜야?" 댓글 → 클릭 → 설치
```

**주간 콘텐츠 캘린더**

| 요일 | 콘텐츠 유형 | 채널 |
|---|---|---|
| 월 | 이번 주 소원 공개 (감성 카드) | 인스타, 카카오 |
| 수 | 당첨자 인터뷰 / 반응 영상 | 유튜브 쇼츠, X |
| 금 | 블록 해시 인증 교육 콘텐츠 | X, 디스코드 |
| 주말 | PRIME 역경매 하이라이트 | 전 채널 |

**ROAS 최적화 구조**

```
광고 집행
    ↓
첫 충전 전환율 모니터링 (목표: 8%+)
    ↓
충전 ARPU 측정 (목표: 5,000원+/월)
    ↓
LTV = ARPU × 평균 이용 기간 (목표: 3만 원+)
    ↓
CAC 상한 = LTV / 3 (약 10,000원)
```

### 5-1. North Star Metric 흐름

|---|---|---|
| Pre-Launch | 사전 등록자 수 | 5,000명 |
| Phase 1 | 월간 추첨 참여 횟수 | 5만 회/월 |
| Phase 2 | 월간 추첨 참여 횟수 | 50만 회/월 |
| Phase 3 | 월간 추첨 참여 횟수 + 광고 수익 | 500만 회/월 + 월 1억 원+ |

### 5-2. 단계별 입력 지표 vs 출력 지표

| 단계 | 입력 지표 (Leading) | 출력 지표 (Lagging) |
|---|---|---|
| Pre-Launch | 인플루언서 수, 커뮤니티 가입자 | 사전 등록자 수, CBT K-Factor |
| Phase 1 | 소원 공유율, K-Factor, CPI | MAU, D7 리텐션, 충전 전환율 |
| Phase 2 | 파트너 수, ROAS, ARPPU | MAU, LTV/CAC, 월 매출 |
| Phase 3 | 브랜드 인지도, 구독 전환율 | MAU 50만, 월 매출 30억 |

### 5-3. 리스크별 대응 시나리오

| 리스크 | 발생 조건 | 대응 액션 |
|---|---|---|
| K-Factor 0.3 미만 | Phase 1 4주 차 | 바이럴 루프 UX 재설계 후 광고 집행 |
| 충전 전환율 3% 미만 | Phase 1 2개월 차 | 첫 충전 인센티브 2배 증가 |
| LTV/CAC 2:1 미만 | Phase 2 6개월 차 | 광고 채널 재배분, 리텐션 기능 강화 |
| 규제 이슈 발생 | 전 단계 | 법무팀 즉시 대응, 게임 방식 조정 |
| 대기업 유사 서비스 출시 | Phase 2~3 | 공정성 캠페인 강화, 커뮤니티 락인 |
