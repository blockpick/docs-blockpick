# 유닛 이코노미 (Unit Economics)

유닛 이코노미는 **유저 한 명을 획득하는 비용(CAC)** 대비 **유저 한 명이 창출하는 가치(LTV)**를 분석하여 비즈니스의 경제적 건전성을 평가하는 핵심 지표 체계입니다.

<Callout type="info">
**목표:** LTV / CAC ≥ 3 (건전한 비즈니스 기준). 블록픽은 바이럴 루프와 무료 포인트 구조로 CAC를 최소화하면서 LTV를 극대화합니다.
</Callout>

---

## 1. 핵심 지표 정의

| 지표 | 정의 | 산출 방법 |
|------|------|----------|
| **CAC** | Customer Acquisition Cost — 유저 1명 획득 비용 | 총 UA 비용 ÷ 신규 설치 수 |
| **LTV** | Lifetime Value — 유저 생애 기대 수익 | ARPU ÷ Churn Rate |
| **ARPU** | Average Revenue Per User — 유저당 월평균 수익 | 월 총 매출 ÷ MAU |
| **ARPPU** | Average Revenue Per Paying User — 결제 유저당 월평균 수익 | 월 결제 매출 ÷ 결제 유저 수 |
| **Payback Period** | CAC 회수 기간 | CAC ÷ 월 ARPU |
| **Churn Rate** | 월 이탈률 | (이전 달 MAU - 이번 달 유지 MAU) ÷ 이전 달 MAU |
| **Retention** | 잔존율 | 1 - Churn Rate |
| **K-Factor** | 바이럴 계수 | 유저당 초대 수 × 초대 수락률 |

---

## 2. LTV 산출 모델

### 2-1. 기본 LTV 공식

```
LTV = ARPU × (1 / Churn Rate)
    = ARPU × 평균 이용 기간(월)
```

### 2-2. 광고 수익 포함 통합 LTV

블록픽은 결제를 하지 않는 무과금 유저도 광고를 통해 수익을 창출합니다. 따라서 광고 수익을 포함한 통합 LTV를 사용합니다.

```
통합 LTV = (ARPU_결제 + ARPU_광고) × (1 / Churn Rate)

여기서:
  ARPU_광고 = 일평균 광고 시청 횟수 × eCPM / 1,000 × 30일
```

**예시 계산 (기본 시나리오):**
```
ARPU_광고    = 7회/일 × ₩8,000/1,000회 × 30일 = ₩1,680/월
ARPU_결제    = (결제 전환율 5%) × ARPPU ₩18,000 = ₩900/월
총 ARPU      = ₩2,580/월
Churn Rate   = 15%/월
LTV          = ₩2,580 ÷ 0.15 = ₩17,200
```

### 2-3. 세그먼트별 LTV

| 세그먼트 | 비율 | 월 ARPU | Churn Rate | LTV |
|---------|------|---------|-----------|-----|
| **무과금 라이트** | 60% | ₩1,200 (광고만) | 25% | ₩4,800 |
| **무과금 헤비** | 20% | ₩2,500 (광고 집중) | 15% | ₩16,700 |
| **소과금** | 12% | ₩6,000 (결제 1~2회/월) | 12% | ₩50,000 |
| **중과금** | 6% | ₩18,000 (결제 3~5회/월) | 8% | ₩225,000 |
| **고과금 (고래)** | 2% | ₩60,000+ (결제 10회+/월) | 5% | ₩1,200,000+ |

**가중 평균 LTV:**
```
가중 평균 LTV = 0.60×₩4,800 + 0.20×₩16,700 + 0.12×₩50,000
              + 0.06×₩225,000 + 0.02×₩1,200,000
             = ₩2,880 + ₩3,340 + ₩6,000 + ₩13,500 + ₩24,000
             = ₩49,720
```

<Callout type="info">
고과금 유저 2%가 전체 LTV의 약 48%를 기여합니다. **고과금 유저 리텐션이 최우선 과제**입니다.
</Callout>

## 3. CAC 시나리오

### 3-1. 오가닉 채널 (K-Factor 기반 바이럴)

블록픽의 VIBE 게임(소문내기)과 당첨 인증 공유는 자연 바이럴을 유도합니다.

```
K-Factor = 유저당 평균 초대 수 × 초대 수락률

유효 CAC_오가닉 = 기존 마케팅 비용 / (신규 유저 × (1 + K / (1 - K)))
```

| K-Factor | 바이럴 배율 | 효과 |
|---------|-----------|------|
| 0.1 | 1.11x | 미미한 바이럴 |
| 0.3 | 1.43x | 완만한 성장 가속 |
| 0.5 | 2.00x | 유의미한 오가닉 확보 |
| 0.8 | 5.00x | 강력한 바이럴 (목표) |

**목표 K-Factor:** 0.4 ~ 0.6 (VIBE + 당첨 인증 활성화 시)

### 3-2. 퍼포먼스 마케팅 (CPI/CPA 기반)

| 채널 | CPI (설치당 비용) | 설치→활성 전환율 | 유효 CAC |
|------|-----------------|----------------|---------|
| Meta Ads | ₩800 ~ ₩1,500 | 40% | ₩2,000 ~ ₩3,750 |
| Google UAC | ₩600 ~ ₩1,200 | 45% | ₩1,333 ~ ₩2,667 |
| TikTok Ads | ₩400 ~ ₩900 | 35% | ₩1,143 ~ ₩2,571 |
| 인플루언서 | ₩1,000 ~ ₩3,000 | 60% | ₩1,667 ~ ₩5,000 |
| ASO (자연 검색) | ₩0 | - | ₩0 |

**평균 블렌디드 CAC 목표:** ₩2,000 ~ ₩5,000

### 3-3. 크로스 프로모션

| 방식 | CAC | 특이사항 |
|------|-----|---------|
| 제휴 앱 교환 | ₩0 ~ ₩500 | 트래픽 교환 협약 |
| 커뮤니티 마케팅 | ₩200 ~ ₩800 | 맘카페, 앱 리뷰 커뮤니티 |
| 오프라인 이벤트 | ₩500 ~ ₩2,000 | 팝업, 박람회 |

## 4. LTV/CAC 비율 목표

```
LTV/CAC 비율 = 총 LTV / 유효 CAC
```

| 비율 | 판단 |
|------|------|
| 1:1 미만 | 위험 — 유저 획득 시 손실 발생 |
| 1:1 ~ 2:1 | 주의 — 개선 필요 |
| 3:1 이상 | 건전 — 투자 확대 가능 |
| 5:1 이상 | 우수 — 공격적 성장 가능 |

### Phase별 LTV/CAC 목표

| Phase | MAU 목표 | 목표 LTV | 목표 CAC | 목표 비율 |
|-------|---------|---------|---------|---------|
| Phase 1 | ~1만 | ₩15,000 | ₩8,000 | 1.9:1 |
| Phase 2 | ~10만 | ₩35,000 | ₩5,000 | 7.0:1 |
| Phase 3 | ~50만 | ₩60,000 | ₩3,000 | 20.0:1 |

<Callout type="info">
Phase 1에서는 LTV/CAC가 3:1 미달이 허용됩니다. 바이럴 루프와 리텐션 강화를 통해 Phase 2에서 목표를 달성하는 것이 핵심입니다.
</Callout>

### Payback Period 목표

| Phase | CAC | 월 ARPU | Payback Period |
|-------|-----|---------|---------------|
| Phase 1 | ₩8,000 | ₩1,200 | 약 6.7개월 |
| Phase 2 | ₩5,000 | ₩2,500 | 약 2.0개월 |
| Phase 3 | ₩3,000 | ₩4,000 | 약 0.75개월 |

## 6. 리텐션 개선 레버

높은 리텐션은 LTV를 직접 향상시키는 가장 강력한 수단입니다.

| 리텐션 레버 | 메커니즘 | 기대 효과 |
|-----------|---------|---------|
| 데일리 무료 포인트 | 매일 앱 방문 유도 | Day 7 리텐션 +15%p |
| 미니게임 30종+ | 게임화(Gamification) | Day 30 리텐션 +10%p |
| 포인트 90일 소멸 | 긴급성(Urgency) 부여 | 재방문율 +20% |
| 당첨 알림 푸시 | 감정적 보상 경험 | Day 14 리텐션 +8%p |
| 친구 초대 보상 | 소셜 앵커 효과 | Churn Rate -3%p |
| PRIME 관심 상품 | 희귀 상품 기대감 | 고과금 리텐션 강화 |
