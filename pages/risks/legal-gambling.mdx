## 관련 법률

### 1. 사행행위 등 규제 및 처벌 특례법

| 항목 | 내용 |
|------|------|
| **핵심 조항** | 제2조 "사행행위" 정의 |
| **정의** | "다수인으로부터 재물이나 재산상의 이익을 모아 우연적 방법으로 득실을 결정하여 재산상의 이익이나 손실을 주는 행위" |
| **벌칙** | 영업으로 사행행위를 한 자: 5년 이하 징역 또는 3천만원 이하 벌금 |

### 2. 게임산업진흥에 관한 법률

| 항목 | 내용 |
|------|------|
| **핵심 조항** | 제28조 (사행성 게임물의 이용 제공 등 금지) |
| **정의** | 베팅이나 배당을 내용으로 하는 게임물, 우연적 방법으로 결과가 결정되는 게임물 |
| **게임물 해당 여부** | 블록픽이 "게임물"로 분류되는지 여부가 핵심 쟁점 |

### 3. 경품류 제공에 관한 불공정거래행위의 유형 및 기준

| 항목 | 내용 |
|------|------|
| **근거** | 공정거래위원회 고시 |
| **경품 상한** | 거래가액의 10배 이내 (일반 경품), 추첨식 경품 총액은 매출액의 1% 이내 |
| **적용 가능성** | 블록픽의 상품 제공이 "경품류 제공"에 해당할 수 있음 |

## 블록픽의 법적 지위

### 핵심 쟁점: "추첨에 의한 경품 제공" vs "사행행위"

```
사행행위 = 재물 모집 + 우연적 방법 + 재산상 득실 결정
         ↓
블록픽  = 참여비 수집 + 블록 추첨 + 상품 제공/미제공
```

**판단의 핵심**은 블록픽이 "재산상 이익의 득실을 결정"하는 구조인지 여부입니다.

## 유리한 논거

| 논거 | 상세 |
|------|------|
| **실물/쿠폰 상품** | 당첨 상품은 실물 또는 쿠폰으로 제공되며, **현금 환급이 불가**함. 재산상 이익을 현금으로 전환할 수 없는 구조 |
| **소액 참여비** | 참여비가 10P(약 10원)부터 시작하는 소액 구조로, 과도한 재산 손실이 구조적으로 제한됨 |
| **무료 참여 경로** | 출석체크, 광고 시청, 미니게임 등 무료로 포인트를 획득하여 참여 가능. 반드시 현금을 지불할 필요 없음 |
| **블록체인 투명성** | Ronin 블록체인의 블록 해시 기반으로 추첨이 이루어져 **결과 조작이 불가능**하고, 누구나 검증 가능 |
| **기술적 요소** | 유저가 직접 블록 번호를 선택하는 행위가 포함되어, 순수한 우연이 아닌 **선택/전략 요소**가 존재 |
| **엔터테인먼트 목적** | 소원/공감/소문 등 소셜 기능이 결합된 엔터테인먼트 서비스로서의 성격 |

## 불리한 논거

| 논거 | 상세 |
|------|------|
| **유료 참여 시 재산 손실** | 유료로 충전한 포인트로 참여 시 미당첨이면 참여비를 잃게 되어, 사실상 **재산 손실**이 발생 |
| **상품의 금전적 가치** | 당첨 상품이 금전적 가치를 가진 실물/쿠폰이므로, "재산상 이익"에 해당할 수 있음 |
| **추첨 요소 존재** | 블록 번호 선택에 전략 요소가 있으나, 최종 결과는 블록 해시에 의해 결정되어 **우연적 방법**으로 볼 여지 |
| **다수의 참여비 수집** | 다수의 유저로부터 참여비를 수집하여 상금 풀을 형성하는 구조 |

### 서비스 설계 측면

| 대응 | 내용 |
|------|------|
| **참여비 상한 설정** | 1회 참여비 상한 (예: 1,000P), 일일 총 참여비 상한 설정 |
| **일일 참여 횟수 제한** | 유저당 일일 게임 참여 횟수 제한 (예: DAILY 1회, VIBE 3회, PRIME 1회) |
| **당첨 확률 공시** | 게임 참여 화면에 당첨 확률을 명확히 표시 (예: "300명 참여 시 당첨 확률 1/300") |
| **무료 참여 비중 확대** | 무료 포인트 획득 경로를 충분히 제공하여, 유료 충전 없이도 핵심 기능 이용 가능 |
| **현금 환급 불가** | 포인트 및 당첨 상품의 현금 환급/양도를 엄격히 금지 |
| **미성년자 보호** | 미성년자 유료 참여 제한, 연령 인증 절차 |
