# 재화 시스템

블록픽의 단일 포인트 재화 체계입니다.

import { Callout } from 'nextra/components'

## 포인트 (Point)

블록픽의 **유일한 재화**입니다. 충전도 가능하고, 활동으로도 획득할 수 있습니다.

### 획득 방법

| 경로 | 설명 |
|------|------|
| **인앱 결제** | 포인트 충전 (유료) |
| **리워드 광고** | 광고 시청 후 포인트 지급 |
| **미니게임** | 미니게임 클리어 보상 |
| **출석체크** | 일일 출석 보상 |
| **활동 보상** | 소원 등록, 공감, 후기 작성 등 |

### 사용처

| 용도 | 설명 |
|------|------|
| **DAILY 게임 참여** | 데일리 게임 참여비 |
| **VIBE 소문내기** | 소원 게임 참여 (10P) |
| **PRIME 역경매 입찰** | 입찰 금액으로 사용 |

### 내부 출처 추적

유저에게는 하나의 잔액으로 보이지만, 백엔드에서는 **유료 충전분**과 **무료 획득분**을 구분하여 기록합니다.

| 출처 | 환불 대상 | 차감 순서 | 소멸 |
|------|-----------|-----------|------|
| **유료 충전** (Paid) | O (전자상거래법) | 나중에 차감 | 무제한 |
| **무료 획득** (Free) | X | 먼저 차감 | 90일 |

<Callout type="warning">
  포인트 차감 시 **무료 획득분을 먼저 소진**합니다. 이를 통해 유료 충전분의 환불 가능 금액을 최대한 보전합니다.
</Callout>

### 거래 유형

| 유형 | 설명 | 포인트 변동 |
|------|------|-------------|
| `deposit` | 포인트 충전 (인앱 결제) | + |
| `refund` | 환불 | - |
| `gameEntry` | 게임 참여비 차감 | - |
| `gameReward` | 게임 당첨 보상 | + |
| `adReward` | 광고 시청 보상 | + |
| `attendance` | 출석체크 보상 | + |
| `activity` | 활동 보상 (공감, 후기 등) | + |
| `miniGame` | 미니게임 클리어 보상 | + |

### 환불 정책

- 유료 충전분만 환불 대상
- 무료 획득 포인트를 하나라도 사용한 경우, 해당 무료분은 환불 불가
- 환불 가능 금액 = 유료 충전 잔액 (무료분 먼저 차감이므로 보전됨)
