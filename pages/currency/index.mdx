# 재화 시스템

블록픽의 3가지 재화 체계입니다.

## 재화 종류

| 재화 | 용도 | 획득 방법 |
|------|------|-----------|
| **쇼핑캐시** (Shopping Cash) | 게임 참여비 결제 | 충전 (인앱 결제) |
| **이벤트 포인트** (Event Points) | 게임 참여 / 보너스 | 게임 참여, 리워드 광고 시청 |
| **참여 포인트** (Participation Points) | 로열티 / 인게이지먼트 | 출석체크, 활동 |

## 쇼핑캐시

게임 참여에 사용하는 **유료 재화**입니다.

- 인앱 결제로 충전합니다 (PG사 연동 필요 - Phase 1)
- 게임 참여 시 차감됩니다
- 환불이 가능합니다

## 이벤트 포인트

활동으로 획득하는 **무료 재화**입니다.

- 게임 참여 시 보너스로 지급됩니다
- 리워드 광고 시청 시 지급됩니다
- 게임 참여비로 사용할 수 있습니다
- 일일 광고 시청 횟수 제한이 있습니다

## 참여 포인트

로열티 프로그램 용 재화입니다.

- 출석체크로 획득합니다
- 각종 활동(후기 작성 등)으로 획득합니다

## 구현 현황

| 기능 | 완성도 | 상태 |
|------|--------|------|
| 잔액 표시 | 80% | 완료 |
| 충전 화면 | 50% | UI만 (PG 미연동) |
| 참여비 차감 | 80% | 완료 |
| 환불 화면 | 50% | UI만 |
| 쿠폰 관리 | 60% | 부분 구현 |
| 리워드 광고 (AdMob) | 80% | 완료 |
