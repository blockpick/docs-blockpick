# 포인트 이코노미 & 리텐션 전략

블록픽 전체 포인트 순환 구조와 유저 체류 전략입니다.

<Callout type="info">

## 1. 포인트 순환 구조 (Economy Loop)

### 1-1. 전체 흐름도

```
[획득] → [축적] → [소비] → [보상] → [재획득]
  ↑                                      ↓
  └──────────── 리텐션 루프 ──────────────┘
```

### 1-2. 획득 경로

| 경로 | 유형 | 전략적 역할 |
|------|------|-------------|
| 인앱 결제 충전 | 유료 | **매출 드라이버** |
| 리워드 광고 시청 | 무료 | **광고 수익 + 체류 유도** |
| 미니게임 클리어 | 무료 | **체류 시간 확보** |
| 출석체크/활동 보상 | 무료 | **일일 리텐션** |
| 미당첨 위로 보상 | 무료 | **이탈 방지** |

### 1-3. 소비 경로

| 경로 | 포인트 차감 | 비고 |
|------|-------------|------|
| DAILY 게임 참여 | 게임별 상이 | 일 1회 무료 |
| VIBE 소문내기 | 10P | 광고 시청으로 무료 참여 가능 |
| PRIME 역경매 입찰 | 입찰 금액 | 1P~상품가 |

## 2. 콘텐츠별 포인트 흐름

### 2-1. DAILY (데일리 게임)

```
[매일 접속] → 무료 참여 1회 → 블록픽 판 참여 → 결과
                                                ├─ 당첨 → 상품 수령 + 후기 보상
                                                └─ 미당첨 → 위로 포인트 + 연속 보너스
```

| 행위 | 포인트 | 비고 |
|------|--------|------|
| 매일 첫 참여 (무료) | 0P | 일 1회 무료 참여권 |
| 추가 참여 | -100P~ | 게임별 상이 |
| 미당첨 위로 | +3P | 매 참여 시 |
| 3일 연속 참여 | +20P | 연속 보너스 |
| 7일 연속 참여 | +50P | 연속 보너스 |
| 당첨 후기 작성 | +30P | 후기당 1회 |

**리텐션 포인트**: 매일 무료 1회 → 미당첨 위로 → 연속 보너스 → **매일 돌아올 이유**

### 2-2. WISH (위시/소원/소문)

```
[소원 등록] → 공감 수집 → 소문내기 게임 개설 → 참여자 유입
     ↓              ↓                              ↓
  +50P 보상     +5P/공감                      10P or 광고 시청
                                                    ↓
                                              당첨 → 등록자 1+1 혜택
                                              미당첨 → +2P 위로
```

| 행위 | 포인트 | 비고 |
|------|--------|------|
| 소원 등록 | +50P | 등록당 1회 |
| 소원 공감 | +5P | 일 최대 20회 (100P/일) |
| 소문내기 참여 | -10P | |
| 소문내기 참여 (광고) | 0P | 광고 시청으로 무료 |
| 미당첨 위로 | +2P | 매 참여 시 |
| 소원 등록자 1+1 당첨 | 동일 상품 추가 | 내 소원의 당첨자 발생 시 |
| 내 소원 공감 목표 달성 | +100P | 게임 개설 보너스 |

**리텐션 포인트**: 소원 등록 → 공감 받기 위해 공유 → 바이럴. 공감 행위에도 보상 → **콘텐츠 생산 + 소비 양쪽에 보상**

### 2-3. PRIME (역경매)

```
[고가 상품] → 입찰 금액 결정 → 포인트 차감 → 결과
                                              ├─ 최저 고유가 당첨 → 상품 수령
                                              └─ 미당첨 → 입찰금 50% 포인트 복구
```

| 행위 | 포인트 | 비고 |
|------|--------|------|
| 입찰 참여 | -입찰금 | 1P~상품가 |
| 미당첨 보상 | +입찰금의 50% | 무료 포인트로 복구 |
| 첫 입찰 보너스 | +30P | 신규 참여 유도 |
| 당첨 후기 작성 | +100P | 고가 상품 후기 가치 높음 |

**리텐션 포인트**: 미당첨 시 50% 복구 → **"잃어도 절반은 남는다"** → 재참여 허들 낮춤

### 2-4. 미니게임 (가챠 포함 30종+) — 포인트 획득 전용

<Callout type="info">
  미니게임은 **포인트를 소비하는 곳이 아니라, 포인트를 얻는 곳**입니다. 무료로 플레이하며 포인트를 획득하는 수급 파이프 역할입니다.
</Callout>

```
[미니게임 선택] → 무료 플레이 → 결과
                                ├─ 클리어 보상 (+3~10P)
                                ├─ 연속 클리어 보너스 (+30P)
                                └─ 광고 시청 시 보상 2배
```

| 행위 | 포인트 | 비고 |
|------|--------|------|
| 플레이 | 무료 | 무제한 |
| 클리어 보상 | +3~10P | 난이도별 차등, 일 최대 50P |
| 광고 시청 부스터 | 보상 x2 | 일 최대 5회 |
| 연속 클리어 (5연승) | +30P | 일 1회 |

**리텐션 포인트**: 완전 무료 + 포인트 획득 전용 → 체류 시간 확보 + 포인트 수급원

## 3. 리워드 광고 전략

### 3-1. 광고 컨텍스트별 설계

| 컨텍스트 | 트리거 시점 | 보상 | 일일 한도 | 전략적 역할 |
|----------|-----------|------|-----------|-------------|
| **출석 부스터** | 출석체크 후 | 출석 보상 x2 | 1회/일 | 매일 첫 접속 유도 |
| **TX 대기 부스터** | 블록체인 트랜잭션 대기 중 | +5~10P | 무제한 | 대기 시간 활용 (이탈 방지) |
| **결과 후 부스터** | 게임 결과 확인 후 | 위로 보상 x2 | 무제한 | 미당첨 실망 완화 |
| **홈 일일 보상** | 홈 화면 배너 | +5P | 5회/일 | 자유 수급 |
| **소문내기 무료** | 소문내기 참여 시 | 참여비 면제 | 3회/일 | 위시 참여 활성화 |
| **데일리 추가** | 데일리 추가 참여 시 | 참여비 면제 | 2회/일 | 데일리 참여 활성화 |

### 3-2. 일일 광고 총량

| 항목 | 횟수 |
|------|------|
| 출석 부스터 | 1회 |
| TX 대기 부스터 | 가변 (평균 2~3회) |
| 결과 후 부스터 | 가변 (참여 횟수 비례) |
| 홈 일일 보상 | 5회 |
| 소문내기 무료 | 3회 |
| 데일리 추가 | 2회 |
| **일일 최소 고정** | **11회** |

### 3-3. 광고 세션 흐름 (구현 기준)

```
1. 세션 시작 (서버에 컨텍스트 + idempotencyKey 전송)
2. 서버에서 verificationToken + 예상 보상 금액 응답
3. AdMob 리워드 광고 표시
4. 유저 시청 완료
5. 세션 완료 (서버에 verificationToken + 광고 결과 전송)
6. 서버 검증 후 포인트 지급
```

<Callout type="warning">
  광고 보상은 서버에서 검증 후 지급합니다. 클라이언트에서 직접 포인트를 증가시키지 않습니다. idempotencyKey로 중복 지급을 방지합니다.
</Callout>

## 6. 당첨 후 재순환 루프

```
당첨 → 상품 수령 → 후기 작성 유도 → 포인트 보상 → 재참여
                                         ↓
                                   다른 유저 후기 열람 → 신뢰도 → 참여
```

| 행위 | 보상 | 조건 |
|------|------|------|
| 후기 작성 (텍스트) | +30P | 당첨 후 7일 이내 |
| 후기 작성 (사진) | +50P | 사진 1장 이상 |
| 후기 작성 (영상) | +100P | 영상 1개 이상 |
| 인기 후기 선정 | +200P | 좋아요 50+ 월간 선정 |
| 당첨 공유 (SNS) | +20P | 1회 한정 |

## 7. 포인트 소멸 정책

| 출처 | 유효기간 | 소멸 알림 | 비고 |
|------|----------|-----------|------|
| **유료 충전** | 무제한 | - | 전자상거래법 환불 보호 |
| **무료 획득** | **90일** | 만료 7일 전 푸시 | 소비 압박 → 활동 유도 |
| 쿠폰 | 30일 | 만료 3일 전 푸시 | 쿠폰별 상이 |

<Callout type="warning">
  무료 획득 포인트의 90일 소멸이 핵심입니다. "쓰지 않으면 사라진다" → 게임 참여 유도 → **자연스러운 활동 압박**.
</Callout>

## 8. 수치 밸런스

### 8-1. 무과금 유저 일일 최대 획득

| 경로 | 포인트 |
|------|--------|
| 출석체크 (+ 광고 부스터) | +20P |
| 데일리 무료 1회 + 위로 | +3P |
| 소원 공감 20회 | +100P |
| 미니게임 클리어 (일 최대) | +50P |
| 미니게임 광고 부스터 5회 | +50P |
| 홈 광고 5회 | +25P |
| **일일 합계** | **~248P** |
| **월간 합계 (28일)** | **~6,944P** |

### 8-2. 포인트 → 게임 참여 전환

| 게임 | 참여비 | 무과금 월간 가능 횟수 |
|------|--------|----------------------|
| DAILY (추가 참여) | 100P | ~69회 |
| VIBE (소문내기) | 10P | ~694회 |
| PRIME (입찰) | 가변 | 입찰 금액에 따라 |

### 8-3. 과금 vs 무과금 밸런스

| 항목 | 무과금 | 과금 |
|------|--------|------|
| 일일 게임 참여 | 무료 1회 + 광고 2~3회 | 무제한 |
| 당첨 확률 | 동일 | 동일 (참여 횟수 차이뿐) |
| PRIME 입찰 | 무료 획득분만 | 자유 입찰 |
| 포인트 소멸 걱정 | 있음 (90일) | 없음 (유료분 무제한) |

<Callout type="info">
  **핵심 원칙**: 과금은 "더 많이 참여"할 수 있게 해줄 뿐, 당첨 확률 자체에 영향 없음. 공정성 = 블록체인 투명성과 일관된 메시지.
</Callout>

```
                    ┌─────────────────────────────────────────┐
                    │              유저 행동                    │
                    └─────────┬───────────────────┬───────────┘
                              │                   │
                    ┌─────────▼─────────┐ ┌──────▼──────────┐
                    │   결제 (인앱)      │ │  활동 (무과금)    │
                    │   → 포인트 충전    │ │  → 광고, 출석,   │
                    │   [유료 출처]      │ │    공감, 미니게임  │
                    │                   │ │    [무료 출처]    │
                    └─────────┬─────────┘ └──────┬──────────┘
                              │                   │
                              └─────────┬─────────┘
                                        │
                              ┌─────────▼─────────┐
                              │    포인트 (P)       │
                              │  유저에겐 하나의 잔액 │
                              └─────────┬─────────┘
                                        │
                    ┌───────────────────┼───────────────────┐
                    │                   │                   │
              ┌─────▼─────┐     ┌──────▼──────┐     ┌─────▼─────┐
              │  DAILY     │     │  WISH       │     │  PRIME    │
              │  데일리게임 │     │  소문내기    │     │  역경매    │
              └─────┬─────┘     └──────┬──────┘     └─────┬─────┘
                    │                   │                   │
              ┌─────▼─────────────────▼───────────────────▼─────┐
              │                      결과                         │
              │  당첨 → 상품 + 후기 보상 (+P)                      │
              │  미당첨 → 위로 보상 (+P) + 광고 부스터 기회          │
              └──────────────────────┬──────────────────────────┘

              ┌──────────────────────▼──────────────────────────┐
              │                  재참여 루프                      │
              │  연속 보너스 / 소멸 압박 / 미니게임 수급            │
              └─────────────────────────────────────────────────┘
```
