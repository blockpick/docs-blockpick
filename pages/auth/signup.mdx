# 회원가입

블록픽의 회원가입 방식과 상세 플로우입니다.

## 가입 방식

| 방식 | 구현 상태 | 플로우 |
|------|-----------|--------|
| **이메일/비밀번호** | 완료 | 이메일 입력 → 인증코드 발송 → 코드 확인 → 비밀번호 설정 → 약관 동의 → 가입 완료 |
| **Google 소셜 로그인** | 완료 | Google OAuth → 전화번호 인증 → 약관 동의 → 가입 완료 |
| **Apple 소셜 로그인** | 완료 | Apple ID 인증 → 전화번호 인증 → 약관 동의 → 가입 완료 |
| **전화번호 인증** | 완료 | SMS 인증코드 → 이메일/비밀번호 설정 → 약관 동의 |

## 이메일 회원가입 (FR-001)

| 항목 | 내용 |
|------|------|
| **우선순위** | 필수 |
| **사용자** | 비회원 → 일반 회원 |
| **입력** | 이메일, 비밀번호, 인증코드 |
| **출력** | 계정 생성, accessToken/refreshToken 발급 |
| **비즈니스 규칙** | 이메일 중복 불가, 비밀번호 최소 8자, 인증코드 유효시간 5분 |
| **검증 기준** | 가입 완료 후 자동 로그인, 토큰 정상 발급 |

## SMS 전화번호 인증 (FR-005)

| 항목 | 내용 |
|------|------|
| **입력** | 전화번호 |
| **출력** | SMS 인증코드 발송, 인증 성공/실패 |
| **비즈니스 규칙** | 1분 내 재발송 불가, 인증코드 5분 유효, 전화번호 1개당 1계정 |

## 약관 동의 (FR-008)

| 항목 | 내용 |
|------|------|
| **설명** | 회원가입 시 이용약관, 개인정보처리방침, 마케팅 수신 동의 |
| **비즈니스 규칙** | 필수 약관 미동의 시 가입 불가, 마케팅 동의는 선택 |

## 비밀번호 찾기/재설정 (FR-007)

| 항목 | 내용 |
|------|------|
| **설명** | 이메일 또는 전화번호로 본인 확인 후 비밀번호 재설정 |
| **입력** | 이메일 또는 전화번호, 인증코드, 새 비밀번호 |
| **비즈니스 규칙** | 이전 3개 비밀번호와 동일 불가 (추후) |

## 회원 탈퇴 (FR-009)

사용자 요청 시 계정을 비활성화합니다. 개인정보보호법에 따라 탈퇴 후 30일간 데이터를 보관한 후 파기합니다.
