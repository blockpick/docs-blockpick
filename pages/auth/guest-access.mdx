# 비회원 참여

파트너 게임에서 비회원(게스트)이 참여하는 방식입니다.

## 참여 조건

- 파트너 브랜드 게임(PARTNER 타입)에서만 비회원 참여가 가능합니다
- 전화번호만 입력하면 참여할 수 있습니다
- 회원가입이 필요하지 않습니다

## 플로우

1. SNS나 외부 링크를 통해 파트너 게임 페이지에 접근
2. 전화번호 입력
3. 가챠 게임 참여
4. 즉시 당첨 여부 확인
5. 당첨 시 기프티쇼 쿠폰 URL을 문자로 발송

## 기술 구현

| 항목 | 내용 |
|------|------|
| **토큰** | `guestToken` 발급 (당첨 확인용) |
| **API** | `joinPartnerGame` Mutation 사용 |
| **입력** | gameId, idempotencyKey, phoneNumber |
| **출력** | entryId, guestToken, prize (당첨 시) |
| **중복 방지** | `idempotencyKey`로 동일 요청 중복 처리 방지 |
