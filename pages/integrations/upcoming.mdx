# 미구현 연동

## PG사 결제 연동

| 항목 | 내용 |
|------|------|
| **용도** | 쇼핑캐시 충전 (인앱 결제) |
| **예상 PG사** | 토스페이먼츠, NHN KCP, 또는 인앱 결제 (Apple IAP / Google Play Billing) |
| **Phase** | Phase 2 |
| **요구사항** | PCI-DSS 준수, 환불 처리 자동화 |
