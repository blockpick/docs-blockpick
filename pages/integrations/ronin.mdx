# Ronin 블록체인

## 연동 정보

| 항목 | 내용 |
|------|------|
| **네트워크** | Ronin Mainnet |
| **Chain ID** | 2020 |
| **프로토콜** | EVM 호환 (Ethereum JSON-RPC) |
| **클라이언트** | web3dart 2.7.3 (Flutter) |
| **연동 방식** | RPC 직접 호출 (앱 → Ronin 노드) |

## 연동 기능

| 기능 | 설명 | 호출 주체 |
|------|------|-----------|
| 지갑 생성 | BIP39 니모닉 → 프라이빗 키 → 지갑 주소 생성 | 앱 (로컬) |
| 암호화 키 요청 | 스마트 컨트랙트에서 암호화 키 발급 | 앱 → API → 컨트랙트 |
| 좌표 암호화/기록 | AES-256 암호화된 좌표를 온체인에 기록 | 앱 → API → 컨트랙트 |
| 트랜잭션 상태 조회 | 엔트리의 온체인 확인 상태 폴링 | 앱 → API |
| 결과 검증 | 트랜잭션 해시로 게임 결과 검증 | 사용자 (블록 탐색기) |

## 보안 고려사항

- 프라이빗 키는 디바이스 Keychain/Keystore에만 저장
- 서버로 프라이빗 키 전송 금지
- 스마트 컨트랙트 감사(Audit) 필요
