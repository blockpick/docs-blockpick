# AWS S3

## 연동 정보

| 항목 | 내용 |
|------|------|
| **버킷** | `blockpick-dev-storage` (개발) |
| **리전** | us-west-2 |
| **접근 방식** | 웹: Next.js API 프록시 (`/api/image-proxy/*`), 앱: 직접 URL |
| **CORS** | 웹에서 직접 접근 시 CORS 이슈 → 프록시로 우회 |

## 사용 용도

| 용도 | 경로 패턴 | 접근 권한 |
|------|-----------|-----------|
| 상품 이미지 | `/products/{gameId}/{imageId}` | Public Read |
| 후기 이미지 | `/reviews/{userId}/{imageId}` | Public Read |
| 관리자 업로드 | `/uploads/{category}/{file}` | Authenticated |
