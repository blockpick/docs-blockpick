# 바이럴/공유 전략

블록픽의 핵심 성장 엔진은 소원 공유입니다. 유저가 소원을 등록하면 자연스럽게 공유 행동이 발생하고, 공유를 통해 유입된 유저가 다시 소원을 등록하는 구조를 설계합니다.

## 1. 바이럴 루프 전체 구조

### 1-1. 핵심 루프

```
[소원 등록]
    ↓ 자동 공유 유도 (등록 완료 직후 바텀시트)
[공유 발생]
    ↓ 공유 링크 클릭 → 랜딩 페이지 진입
[공감 유입]
    ↓ 공감 목표 달성
[소문내기 게임 개설]
    ↓ 참여자 모집 (10P 또는 광고 시청)
[게임 참여]
    ↓ 당첨 발생
[당첨 공유]
    ↓ "나도 당첨되고 싶다" → 새 유저 유입
[신규 소원 등록]
    ↓
(루프 반복)
```

### 1-2. 단계별 전환율 예상치 및 최적화 포인트

| 단계 | 전환율 (초기 목표) | 전환율 (6개월 목표) | 핵심 최적화 포인트 |
|------|-------------------|--------------------|--------------------|
| 소원 등록 → 공유 발생 | 40% | 65% | 등록 완료 직후 공유 바텀시트 자동 표시 |
| 공유 링크 클릭 → 앱 진입 | 25% | 40% | 공유 카드 품질, OG 이미지, 딥링크 |
| 앱 진입 → 공감 | 60% | 75% | 랜딩 화면 최적화, 원터치 공감 |
| 앱 진입 → 회원가입 | 30% | 45% | 소셜 로그인 원터치, 가입 인센티브 |
| 회원가입 → 소문내기 참여 | 50% | 65% | 웰컴 패키지 500P 즉시 지급 |
| 당첨 → 당첨 공유 | 55% | 70% | 당첨 바텀시트 자동 표시, 당첨 카드 품질 |
| 당첨 공유 클릭 → 신규 가입 | 20% | 35% | "나도 참여" CTA, 상품 이미지 임팩트 |

<Callout type="warning">
  **전환율 가정 근거**: 국내 유사 커머스 게임화 앱(럭키백, 캐시워크 공유 기능) 벤치마크 기반 초기 목표치입니다. 실제 수치는 런칭 후 2주 데이터로 재설정 필요합니다.
</Callout>

### 1-3. 루프별 K-Factor 목표

```
K-Factor = 공유율 × 클릭률 × 전환율

초기: 0.40 × 0.25 × 0.30 = 0.03 (보조 성장)
6개월: 0.65 × 0.40 × 0.45 = 0.117 (바이럴 보조 성장 궤도)
목표: 0.50 이상 (유료 획득 유저당 0.5명 이상 자연 유입)
```

## 2. 공유 시나리오별 채널 전략

### 2-1. 카카오톡 공유 (최우선 채널)

**이유**: 국내 MAU 4,700만, 메시지 클릭률이 SNS 대비 3~5배 높음. 1:1 및 오픈채팅 모두 활용 가능.

#### 공유 포맷

| 항목 | 스펙 |
|------|------|
| 카드 이미지 크기 | 800 × 400px (2:1 비율, KakaoTalk Feed 표준) |
| 제목 | 최대 25자 |
| 설명 | 최대 60자 |
| 버튼 CTA | "공감하러 가기" / "나도 참여" |

#### 소원 공유 카드 문구 (시나리오별)

**공감 요청형** (공감 목표 미달성 시):
```
제목: 이 상품 갖고 싶어요 🙏
설명: [상품명] 소원에 공감해주시면 추첨 게임이 열려요!
      현재 47/100 공감 달성중
버튼: 공감하러 가기 →
```

**게임 개설형** (공감 목표 달성 후):
```
제목: 추첨 게임 열렸어요! 🎰
설명: [상품명] 소문내기 게임 참여하세요
      10P로 참여 가능 (광고 시청 시 무료)
버튼: 지금 참여하기 →
```

**당첨 공유형**:
```
제목: 저 당첨됐어요! 🎉
설명: [상품명] 블록픽에서 실제로 당첨됐습니다
      나도 도전해볼 수 있어요
버튼: 나도 참여하기 →
```

#### 구현 방식
- Kakao SDK `createDefaultFeed()` 사용
- `templateId` 3종 관리: 공감요청/게임개설/당첨공유
- 공유 완료 콜백에서 +20P 지급 처리 (서버 검증 후)
- 딥링크: `blockpick://wish/{wishId}` → 앱 미설치 시 스토어로 분기

## 3. 공유 인센티브 시스템

### 3-1. 공유 보상 구조

| 행위 | 보상 | 조건 | 한도 |
|------|------|------|------|
| 소원 공유 (카카오/인스타/X) | +20P | 공유 완료 콜백 수신 | 소원당 채널별 1회 |
| URL 복사 | +10P | 복사 버튼 탭 | 소원당 1회 |
| 공유를 통한 친구 가입 | +100P | 친구 첫 게임 참여 완료 시 | 월 10명 한도 |
| 공유를 통한 친구 가입 (친구) | +100P | 가입 완료 + 첫 게임 참여 시 | 1회 |
| 당첨 공유 | +30P | 당첨 후 24시간 이내 공유 | 당첨건당 1회 |
| 당첨 공유 + 후기 작성 | +50P 추가 | 사진 포함 후기 | 당첨건당 1회 |

<Callout type="info">
  **설계 원칙**: 공유 보상은 공유 완료 시 즉시 지급하지 않고, 서버에서 플랫폼 콜백을 수신한 후 지급합니다. URL 복사는 서버 검증이 불가하므로 보상을 낮게 설정합니다.
</Callout>

### 3-2. 추천 코드 시스템

#### 코드 구조

```
추천 코드: 6자리 영숫자 (예: MINSOO7)
생성 시점: 회원가입 완료 직후 자동 생성
유저 선택: 기본값은 자동 생성, 마이페이지에서 1회 변경 가능
```

#### 추천 코드 흐름

```
[추천인] 코드 공유
    ↓
[신규 유저] 앱 설치 → 가입 화면에서 코드 입력 (선택)
    ↓
[신규 유저] 첫 게임 참여 완료
    ↓
[양쪽] +100P 동시 지급 + 푸시 알림
    "OOO님이 추천 코드로 가입하여 100P가 적립되었습니다!"
```

#### 추천 코드 입력 위치

1. 앱 가입 플로우 마지막 단계 (선택 입력, 건너뛰기 가능)
2. 공유 링크 진입 시 URL 파라미터로 자동 적용 (`?code=MINSOO7`)
3. 마이페이지 > 친구 초대 화면에서 직접 입력

#### 추천 코드 통계 (마이페이지 노출)

```
내 추천 코드: MINSOO7  [복사] [공유]

초대한 친구: 3명
받은 보상: 300P

[친구 초대 더 하기 →]
```

### 3-3. 인센티브 남용 방지

| 리스크 | 방지 방법 |
|--------|-----------|
| 자기 추천 (본인 다계정) | 기기 ID + IP + 전화번호 중복 체크 |
| 공유 콜백 위조 | 서버사이드 플랫폼 API 검증 |
| 동일인 반복 공유 | 소원당 채널별 1회 한도, idempotencyKey |
| 허위 친구 가입 | 첫 게임 참여 완료 시에만 보상 지급 |

| 블록체인 인증 | "블록체인 검증 완료 ✓" (신뢰 요소) |
| CTA 버튼 | "나도 참여하기" |
| 하단 | "블록픽 - 공정한 추첨" |

#### 인스타그램 당첨 스토리 카드

```
배경: 상품 이미지 (블러 처리) + 골든 그라디언트 오버레이
중앙: 상품 이미지 (클리어, 카드 형태)
상단: 🎉 당첨! 텍스트 (애니메이션 GIF)
하단: "나도 참여" 링크 스티커
```

### 4-3. 당첨 후기 + 공유 결합 인센티브

```
기본 당첨 공유: +30P
당첨 후기 (텍스트): +30P
당첨 후기 (사진 포함): +50P
당첨 후기 (영상 포함): +100P
후기 공유 (SNS 게재): +20P

최대 조합: 당첨 공유 + 영상 후기 + 공유 = +150P
```

<Callout type="info">
  **후기 공유 최대 인센티브**: 영상 후기를 SNS에 올리면 총 150P 획득 가능. 소문내기 15회 참여 가능한 금액. 콘텐츠 생산 유도의 핵심 레버.
</Callout>

### 4-4. 당첨자 후기 피드 (신뢰 루프)

당첨 후기는 단순 리텐션 보상이 아니라 신규 유저의 신뢰를 형성하는 핵심 콘텐츠입니다.

```
후기 피드 노출 위치:
- 홈 화면 "실제 당첨 후기" 섹션 (상위 3개)
- 소문내기 게임 상세 화면 하단 (동일 카테고리 후기)
- 공유 링크 랜딩 페이지 하단 (신뢰 강화)
- 앱스토어 스크린샷 (선정 후기 활용)
```

## 5. 성장 지표 (KPI Tree)

```
핵심 목표: 월간 신규 유저 중 바이럴 유입 비율 30% 이상
    │
    ├── 공유 발생률 (소원 등록 대비)
    │     ├── 소원 등록 후 공유 바텀시트 노출률
    │     ├── 바텀시트 → 공유 완료 전환율 (목표: 60%)
    │     └── 채널별 공유 비율 (카카오 > 인스타 > X > URL)
    │
    ├── 공유 링크 성과
    │     ├── 공유 링크 클릭률 (목표: 25%)
    │     ├── 클릭 → 앱 진입 전환율
    │     └── 앱 진입 → 가입 전환율 (목표: 30%)
    │
    ├── 추천 코드 성과
    │     ├── 추천 코드 입력율 (신규 가입 중)
    │     ├── 추천인당 월평균 초대 수
    │     └── 초대 → 첫 게임 참여 전환율
    │
    └── 당첨 공유 성과
          ├── 당첨 직후 공유 바텀시트 → 공유 완료율 (목표: 55%)
          ├── 당첨 공유 → 클릭률
          └── 당첨 공유 → 신규 가입 전환율 (목표: 20%)
```
