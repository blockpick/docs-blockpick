# 디자인 특징

블록픽의 디자인 패턴과 특징입니다.

## UI 패턴

| 패턴 | 설명 |
|------|------|
| 라운드 코너 | 2xl, xl 라운딩 |
| 소프트 섀도우 | 부드러운 그림자 효과 |
| 그라디언트 버튼 | `btn-gradient` 클래스 |
| 글래스모피즘 | backdrop-blur 기반 투명 효과 |

## 애니메이션

| 플랫폼 | 라이브러리 |
|--------|-----------|
| **웹** | Framer Motion 12.35 |
| **앱** | Flutter Animate 4.3 |

## 웹 스타일링

- **CSS 프레임워크**: Tailwind CSS 4
- **전역 스타일**: `globals.css` (테마 변수, 유틸리티 클래스)
- **컴포넌트**: Radix UI 기반 (웹)
